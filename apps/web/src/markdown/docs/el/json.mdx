# JSON

Η Languine υποστηρίζει αρχεία JSON ως πηγή μορφής με τις κοινές απαιτήσεις για πλουραλισμό και παρεμβολή. Αυτή η μορφή χρησιμοποιείται σε πολλά πλαίσια όπως το React, το Vue, το Ruby on Rails και άλλα.


---


## Ρύθμιση

Αρχικά, βεβαιωθείτε ότι έχετε ένα αρχείο ρύθμισης languine.json ή languine.ts στη ρίζα του έργου σας. Ακολουθεί ένα παράδειγμα:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Μετάφραση

Με τη ρύθμιση σας έτοιμη, εκτελέστε:

```bash
npx languine@latest translate
```

Όταν εκτελέσετε αυτή την εντολή, η Languine θα:

- Φορτώσει το αρχείο JSON πηγής σας (π.χ., locales/en.json)
- Ανιχνεύσει οποιαδήποτε νέα ή τροποποιημένα κείμενα μετάφρασης
- Δημιουργήσει μεταφράσεις για τις γλώσσες στόχους σας
- Δημιουργήσει ή ενημερώσει τα αρχεία JSON της γλώσσας στόχου (π.χ., locales/sv.json, locales/de.json, locales/fr.json)
