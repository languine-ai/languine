# CI/CD

Η Languine παρέχει μια ενσωμάτωσή CI/CD για να σας βοηθήσει να μεταφράσετε την εφαρμογή σας.

---

## Πώς Λειτουργεί

Η ροή εργασίας CI/CD στην Languine ακολουθεί μια δομημένη διαδικασία για να αυτοματοποιήσει τις ενημερώσεις μετάφρασης στην εφαρμογή σας:

### 1. Αρχικοποίηση Δράσης
- Η ροή εργασίας ξεκινά όταν ενεργοποιείται η δράση
- Ρυθμίζει την απαραίτητη διαμόρφωση Git και τα διαπιστευτήρια

### 2. Επεξεργασία Μηχανής Languine
- Η μηχανή Languine επεξεργάζεται το περιεχόμενο της εφαρμογής σας
- Διαχειρίζεται τις εργασίες μετάφρασης χρησιμοποιώντας τις ρυθμισμένες ρυθμίσεις

### 3. Ανίχνευση Αλλαγών
Το σύστημα ελέγχει για τυχόν αλλαγές στη μετάφραση και ακολουθεί μία από τις δύο διαδρομές:

**Όταν Βρεθούν Αλλαγές:**
- Οι αλλαγές καταχωρούνται στο αποθετήριο
- Οι αλλαγές προωθούνται στο καθορισμένο κλάδο
- Δημιουργείται ένα Pull Request (PR)

**Όταν Δεν Βρεθούν Αλλαγές:**
- Η ροή εργασίας ολοκληρώνεται χωρίς να δημιουργηθεί PR

### 4. Δημιουργία Pull Request
Όταν ανιχνεύονται αλλαγές, δημιουργείται ένα Pull Request με:

**Περιεχόμενο PR:**
- Καθαρός τίτλος που υποδεικνύει ενημερώσεις μετάφρασης
- Λεπτομερής περιγραφή των αλλαγών
- Επιβεβαιώσεις ελέγχου ποιότητας
- Οδηγίες για ανασκόπηση
- Συνδέσμους προς τεκμηρίωση και υποστήριξη

**Η Περιγραφή PR Περιλαμβάνει:**
- Περίληψη των ενημερώσεων μετάφρασης
- Λίστα με το τι άλλαξε (ενημερωμένες μεταφράσεις, συνέπεια ορολογίας)
- Επόμενα βήματα για τους ανακριτές
- Συνδέσμους προς πόρους τεκμηρίωσης και υποστήριξης
