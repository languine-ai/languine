# Μάρκα

Ένα μεταφρασμένο αρχείο markdown θεωρείται Έγγραφο και μεταφράζεται ως σύνολο.

## Αγνοώντας περιεχόμενο

Μπορείτε να αγνοήσετε το περιεχόμενο μεταξύ των ετικετών `<!-- languine-ignore-start -->` και `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Αυτό το περιεχόμενο δεν θα μεταφραστεί.
<!-- languine-ignore-end -->
```
