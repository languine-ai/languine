# Αυθεντικοποίηση

Για να χρησιμοποιήσετε το Languine, πρέπει να αυθεντικοποιηθείτε με την Engine μας, χρειάζεται μόνο μερικά δευτερόλεπτα για να ξεκινήσετε.

---

## Σύνδεση

Μπορείτε να συνδεθείτε απευθείας στο [Languine Dashboard](https://languine.ai/login) ή να εκτελέσετε την παρακάτω εντολή:

```bash title="Terminal"
npx languine@latest auth login
```


Αν έχετε ήδη λογαριασμό, θα συνδεθείτε αυτόματα και μια συνεδρία θα αποθηκευτεί στη συσκευή σας.

---

## Αποσύνδεση

Μπορείτε να αποσυνδεθείτε εκτελώντας την παρακάτω εντολή:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Συνεδρία χρήστη

Μπορείτε να ελέγξετε αν είστε συνδεδεμένοι εκτελώντας την παρακάτω εντολή:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Μεταβλητές Περιβάλλοντος

Μπορείτε να ορίσετε τις παρακάτω μεταβλητές περιβάλλοντος για να διαμορφώσετε το Languine:

- `LANGUINE_API_KEY`: Το API key που θα χρησιμοποιήσετε για την αυθεντικοποίηση.
- `LANGUINE_PROJECT_ID`: Το project ID που θα χρησιμοποιήσετε για το έργο.

Μπορείτε να τα βρείτε στο [Languine Dashboard](https://languine.ai/login).
