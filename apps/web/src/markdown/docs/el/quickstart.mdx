# Γρήγορη Εκκίνηση

Ξεκινήστε εγκαθιστώντας το Languine.

Στη συνέχεια, εκτελέστε την παρακάτω εντολή για να αρχικοποιήσετε το Languine.

```bash title="Τερματικό"
npx languine@latest init
```

Ακολουθήστε τα βήματα για να ρυθμίσετε το έργο σας.

## Πρώτη Μετάφραση

Τώρα είστε έτοιμοι να μεταφράσετε τον κώδικά σας.

Ξεκινήστε προσθέτοντας ένα κλειδί στο αρχείο πηγής σας, για παράδειγμα αν επιλέξετε να χρησιμοποιήσετε το `en.json` ως αρχείο πηγής, προσθέστε ένα κλειδί σε αυτό.

```json title="src/locales/en.json"
{
  "welcome": "Καλώς ήρθατε στην εφαρμογή μας!",
  "user": {
    "greeting": "Γειά σου, {{name}}!",
    "profile": {
      "title": "Το προφίλ σας",
      "edit": "Επεξεργασία προφίλ"
    }
  },
  "notifications": {
    "messages": "Έχετε {{count}} νέο(α) μήνυμα(τα).",
    "empty": "Δεν έχετε νέες ειδοποιήσεις."
  }
}

```


## Μετάφραση

Εκτελέστε την παρακάτω εντολή για να μεταφράσετε το περιεχόμενό σας:

```bash title="Τερματικό"
npx languine@latest translate
```

## Αυτό θα:
1. Δημιουργήσει αρχεία μετάφρασης για κάθε γλώσσα στόχο
2. Δημιουργήσει υψηλής ποιότητας μεταφράσεις με το Languine Engine
3. Δημιουργήσει ένα αρχείο `languine.lock` για να παρακολουθεί τις καταστάσεις μετάφρασης

Το αρχείο κλειδώματος βοηθά το Languine να αναγνωρίζει ποιες μεταφράσεις χρειάζονται ενημερώσεις όταν αλλάζει το περιεχόμενο πηγής σας. Βεβαιωθείτε ότι θα δεσμεύσετε αυτό το αρχείο στο αποθετήριο σας.

## Επόμενα Βήματα

Τώρα που έχετε ρυθμίσει το Languine, μπορείτε να:
- Προσθέσετε περισσότερες γλώσσες στο αρχείο `languine.json`
- Ενσωματώσετε το Languine στην CI/CD ροή εργασιών σας
- Χρησιμοποιήσετε το API μας για προγραμματισμένες μεταφράσεις

Δείτε τους άλλους οδηγούς μας για να μάθετε περισσότερα σχετικά με τις προηγμένες δυνατότητες του Languine.
