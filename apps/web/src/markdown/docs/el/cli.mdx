# CLI

Η Languine παρέχει μια CLI για να σας βοηθήσει να μεταφράσετε την εφαρμογή σας.

## Επισκόπηση Εντολών

### translate

Η εντολή `translate` χρησιμοποιείται για να μεταφράσετε την εφαρμογή σας.

```bash title="Terminal"
# Ξεκινήστε τη μετάφραση της εφαρμογής σας
languine translate

# Επιλογές:
#   --source-locale    Καθορίστε τη γλώσσα προέλευσης (προεπιλογή: en)
#   --target-locale    Καθορίστε τη γλώσσα προορισμού(ες) για μετάφραση
#   --config          Διαδρομή προς το αρχείο σας languine.config.js
```

### init

```bash title="Terminal"
# Αρχικοποιήστε ένα νέο έργο Languine
languine init

# Αυτό θα:
# - Δημιουργήσει ένα αρχείο languine.config.js
# - Ρυθμίσει τη δομή του έργου σας
# - Αρχικοποιήσει τις απαραίτητες καταλόγους
```

### status

```bash title="Terminal"
# Ελέγξτε την κατάσταση των μεταφράσεών σας
languine status

# Δείχνει:
# - Τρέχουσα πρόοδο μετάφρασης
# - Ελλείπουσες μεταφράσεις
# - Στατιστικά μετάφρασης
```

### validate

```bash title="Terminal"
# Επικυρώστε τα αρχεία μετάφρασής σας
languine validate

# Ελέγχει:
# - Δομή αρχείου
# - Ελλείπουσες κλειδιά
# - Συνοχή μορφής
```

### export

```bash title="Terminal"
# Εξάγετε τις μεταφράσεις σας
languine export

# Επιλογές:
#   --format          Μορφή εξαγωγής (json, yaml, κ.λπ.)
#   --output         Κατάλογος εξόδου
```

### import

```bash title="Terminal"
# Εισάγετε μεταφράσεις από εξωτερικά αρχεία
languine import

# Επιλογές:
#   --source         Αρχείο ή κατάλογος προέλευσης
#   --format         Μορφή εισαγωγής (json, yaml, κ.λπ.)
```

## Παγκόσμιες Επιλογές

Όλες οι εντολές υποστηρίζουν τις παρακάτω παγκόσμιες επιλογές:

```bash title="Terminal"
# Παγκόσμιες επιλογές διαθέσιμες για όλες τις εντολές
languine [εντολή] [επιλογές]

# Κοινές επιλογές:
#   --help, -h       Εμφάνιση πληροφοριών βοήθειας
#   --version, -v    Εμφάνιση πληροφοριών έκδοσης
#   --verbose        Ενεργοποίηση λεπτομερούς καταγραφής
#   --config         Καθορίστε τη διαδρομή του αρχείου ρυθμίσεων
```
