# GitHub Actions

Με την ενέργεια github, μπορείτε εύκολα να μεταφράσετε τον κώδικά σας απλώς σπρώχνοντας τον κώδικά σας στο αποθετήριο, η ενέργεια github θα μεταφράσει τα νέα σας κλειδιά, είτε θα δημιουργήσει ένα pull request είτε θα δεσμεύσει τις αλλαγές σας.
Αυτό σας εξοικονομεί πολύ χρόνο και προσπάθεια διατηρώντας παράλληλα υψηλής ποιότητας μεταφράσεις.

---

## Εγκατάσταση

Για να χρησιμοποιήσετε την ενέργεια Languine GitHub στο αποθετήριο σας, θα χρειαστεί να:

1. Ρυθμίσετε τα μυστικά του αποθετηρίου σας με τα διαπιστευτήρια Languine σας
2. Δημιουργήσετε ένα αρχείο ροής εργασίας στο αποθετήριο σας

### Ρύθμιση Μυστικών Αποθετηρίου

Προσθέστε τα παρακάτω μυστικά στις ρυθμίσεις του αποθετηρίου σας (Ρυθμίσεις > Μυστικά και μεταβλητές > Ενέργειες):

- `LANGUINE_API_KEY`: Το κλειδί API Languine σας
- `LANGUINE_PROJECT_ID`: Το ID έργου Languine σας

Μπορείτε να βρείτε αυτές τις τιμές στον πίνακα ελέγχου Languine σας κάτω από τις Ρυθμίσεις Έργου.



### Μεταβλητές Περιβάλλοντος

Η ενέργεια GitHub χρησιμοποιεί τις παρακάτω μεταβλητές περιβάλλοντος:
- `LANGUINE_API_KEY` (Απαιτείται): Το κλειδί API Languine σας για αυθεντικοποίηση
- `LANGUINE_PROJECT_ID` (Απαιτείται): Το ID έργου Languine σας 
- `GITHUB_TOKEN` (Απαιτείται): Παρέχεται αυτόματα από το GitHub, χρησιμοποιείται για τη δημιουργία PRs

### Παράδειγμα Ροής Εργασίας

Δημιουργήστε ένα νέο αρχείο .github/workflows/languine.yml στο αποθετήριο σας με το παρακάτω περιεχόμενο:

```ts title=".github/workflows/release.yml"
name: Ανάπτυξη Ιστοσελίδας

on:
  push:
    branches:
      - '**'
    paths:
      - apps/web/**

jobs:
  translate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
      pull-requests: write
    steps:
      - uses: actions/checkout@v4
      
      - name: Languine Action
        uses: languine/languine-action@v1
        env:
           GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        with:
          api-key: ${{ secrets.LANGUINE_API_KEY }}
          project-id: ${{ secrets.LANGUINE_PROJECT_ID }}
```
 
 Αυτή η ροή εργασίας θα:
- Εκτελείται σε σπρώξιμο προς την κύρια κλάδο
- Εκτελείται σε pull requests προς την κύρια κλάδο
- Μπορεί να ενεργοποιηθεί χειροκίνητα από την καρτέλα Ενεργειών
- Ελέγχει τον κώδικά σας και εκτελεί τη διαδικασία μετάφρασης Languine
- Δημιουργεί ένα pull request με οποιεσδήποτε νέες μεταφράσεις (ρυθμιζόμενο)

#### Επιλογή Pull Request

Η επιλογή `create-pull-request` ελέγχει εάν η ενέργεια δημιουργεί ένα pull request με τις αλλαγές ή τις δεσμεύει απευθείας στην κλάδο:

```typescript title=".github/workflows/languine.yml"
with:
  create-pull-request: true # δημιουργία ενός pull request με τις αλλαγές
```

Όταν ρυθμιστεί σε `true`, η ενέργεια θα:
- Δημιουργήσει μια νέα κλάδο με τις αλλαγές
- Δημιουργήσει ένα pull request για αναθεώρηση
- Προσθέσει κατάλληλες ετικέτες και περιγραφή

Όταν ρυθμιστεί σε `false` (προεπιλογή), η ενέργεια θα:
- Δεσμεύσει τις αλλαγές απευθείας στην τρέχουσα κλάδο
- Σπρώξει τις αλλαγές χωρίς να δημιουργήσει ένα pull request
