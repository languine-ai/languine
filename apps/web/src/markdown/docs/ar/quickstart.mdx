# بداية سريعة

ابدأ بتثبيت Languine.

ثم، قم بتشغيل الأمر التالي لتهيئة Languine.

```bash title="Terminal"
npx languine@latest init
```

اتبع الخطوات لإعداد مشروعك.

## الترجمة الأولى

الآن أنت جاهز لترجمة الكود الخاص بك.

ابدأ بإضافة مفتاح إلى ملف المصدر الخاص بك، على سبيل المثال إذا اخترت استخدام `en.json` كملف مصدر، أضف مفتاحًا إليه.

```json title="src/locales/en.json"
{
  "welcome": "مرحبًا بك في تطبيقنا!",
  "user": {
    "greeting": "مرحبًا، {{name}}!",
    "profile": {
      "title": "ملفك الشخصي",
      "edit": "تعديل الملف الشخصي"
    }
  },
  "notifications": {
    "messages": "لديك {{count}} رسالة(رسائل) جديدة.",
    "empty": "لا توجد إشعارات جديدة."
  }
}

```


## الترجمة

قم بتشغيل الأمر التالي لترجمة المحتوى الخاص بك:

```bash title="Terminal"
npx languine@latest translate
```

## هذا سيفعل:
1. إنشاء ملفات ترجمة لكل لغة مستهدفة
2. توليد ترجمات عالية الجودة باستخدام محرك Languine
3. إنشاء ملف `languine.lock` لتتبع حالات الترجمة

يساعد ملف القفل Languine في تحديد الترجمات التي تحتاج إلى تحديثات عندما يتغير محتوى المصدر الخاص بك. تأكد من الالتزام بهذا الملف في مستودعك.

## الخطوات التالية

الآن بعد أن قمت بإعداد Languine، يمكنك:
- إضافة المزيد من اللغات في ملف `languine.json`
- دمج Languine في خط أنابيب CI/CD الخاص بك
- استخدام واجهة برمجة التطبيقات الخاصة بنا للترجمات البرمجية

اطلع على أدلتنا الأخرى لمعرفة المزيد عن الميزات المتقدمة لـ Languine.
