# المصادقة

لاستخدام Languine، تحتاج إلى المصادقة مع محركنا، يستغرق الأمر بضع ثوانٍ للبدء.

---

## تسجيل الدخول

يمكنك تسجيل الدخول مباشرة على [لوحة تحكم Languine](https://languine.ai/login) أو تشغيل الأمر التالي:

```bash title="Terminal"
npx languine@latest auth login
```


إذا كان لديك حساب بالفعل، فسيتم تسجيل دخولك تلقائيًا وسيتم حفظ جلسة على جهازك.

---

## تسجيل الخروج

يمكنك تسجيل الخروج عن طريق تشغيل الأمر التالي:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## جلسة المستخدم

يمكنك التحقق مما إذا كنت مسجلاً للدخول عن طريق تشغيل الأمر التالي:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# متغيرات البيئة

يمكنك تعيين متغيرات البيئة التالية لتكوين Languine:

- `LANGUINE_API_KEY`: مفتاح API لاستخدامه في المصادقة.
- `LANGUINE_PROJECT_ID`: معرف المشروع لاستخدامه في المشروع.

يمكنك العثور عليها في [لوحة تحكم Languine](https://languine.ai/login).
