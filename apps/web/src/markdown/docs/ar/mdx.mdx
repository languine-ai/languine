## تجاهل المحتوى

يمكنك تجاهل المحتوى بين علامات `<!-- languine-ignore-start -->` و `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
هذا المحتوى لن يتم ترجمته.
<!-- languine-ignore-end -->
```
