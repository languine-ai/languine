# CI/CD

تقدم Languine تكامل CI/CD لمساعدتك في ترجمة تطبيقك.

---

## كيف يعمل

يتبع سير العمل في CI/CD في Languine عملية منظمة لأتمتة تحديثات الترجمة في تطبيقك:

### 1. بدء الإجراء
- يبدأ سير العمل عندما يتم تفعيل الإجراء
- يقوم بإعداد تكوين Git والبيانات اللازمة

### 2. معالجة محرك Languine
- يقوم محرك Languine بمعالجة محتوى تطبيقك
- يتولى مهام الترجمة باستخدام الإعدادات المكونة

### 3. كشف التغييرات
يتحقق النظام من أي تغييرات في الترجمة ويتبع أحد مسارين:

**عند العثور على تغييرات:**
- يتم الالتزام بالتغييرات في المستودع
- يتم دفع التغييرات إلى الفرع المحدد
- يتم إنشاء طلب سحب (PR)

**عند عدم العثور على تغييرات:**
- يكتمل سير العمل دون إنشاء PR

### 4. إنشاء طلب سحب
عند اكتشاف تغييرات، يتم إنشاء طلب سحب مع:

**محتوى PR:**
- عنوان واضح يشير إلى تحديثات الترجمة
- وصف تفصيلي للتغييرات
- تأكيدات فحص الجودة
- تعليمات للمراجعة
- روابط إلى الوثائق والدعم

**وصف PR يتضمن:**
- ملخص لتحديثات الترجمة
- قائمة بما تم تغييره (ترجمات محدثة، اتساق المصطلحات)
- الخطوات التالية للمراجعين
- روابط إلى الوثائق وموارد الدعم
