# واجهة سطر الأوامر

تقدم Languine واجهة سطر الأوامر لمساعدتك في ترجمة تطبيقك.

## نظرة عامة على الأوامر

### الترجمة

يستخدم أمر `translate` لترجمة تطبيقك.

```bash title="Terminal"
# بدء ترجمة تطبيقك
languine translate

# الخيارات:
#   --source-locale    تحديد اللغة المصدر (افتراضي: en)
#   --target-locale    تحديد اللغة المستهدفة (اللغات) للترجمة إليها
#   --config          مسار ملف languine.config.js الخاص بك
```

### التهيئة

```bash title="Terminal"
# تهيئة مشروع Languine جديد
languine init

# سيقوم هذا بـ:
# - إنشاء ملف languine.config.js
# - إعداد هيكل مشروعك
# - تهيئة الدلائل اللازمة
```

### الحالة

```bash title="Terminal"
# التحقق من حالة الترجمات الخاصة بك
languine status

# يظهر:
# - تقدم الترجمة الحالي
# - الترجمات المفقودة
# - إحصائيات الترجمة
```

### التحقق

```bash title="Terminal"
# التحقق من ملفات الترجمة الخاصة بك
languine validate

# يتحقق من:
# - هيكل الملف
# - المفاتيح المفقودة
# - اتساق التنسيق
```

### التصدير

```bash title="Terminal"
# تصدير الترجمات الخاصة بك
languine export

# الخيارات:
#   --format          تنسيق التصدير (json، yaml، إلخ)
#   --output         دليل الإخراج
```

### الاستيراد

```bash title="Terminal"
# استيراد الترجمات من ملفات خارجية
languine import

# الخيارات:
#   --source         ملف أو دليل المصدر
#   --format         تنسيق الاستيراد (json، yaml، إلخ)
```

## الخيارات العالمية

تدعم جميع الأوامر الخيارات العالمية التالية:

```bash title="Terminal"
# الخيارات العالمية المتاحة لجميع الأوامر
languine [command] [options]

# الخيارات الشائعة:
#   --help, -h       عرض معلومات المساعدة
#   --version, -v    عرض معلومات الإصدار
#   --verbose        تمكين تسجيل مفصل
#   --config         تحديد مسار ملف التكوين
```
