# CLI

Languine pakub CLI-d, et aidata teil oma rakendust tõlkida.

## Käskude Ülevaade

### translate

`translate` käsku kasutatakse teie rakenduse tõlkimiseks.

```bash title="Terminal"
# Alustage oma rakenduse tõlkimist
languine translate

# Valikud:
#   --source-locale    Määrake lähtekeel (vaikimisi: en)
#   --target-locale    Määrake sihtkeel(d), kuhu tõlkida
#   --config          Teie languine.config.js faili tee
```

### init

```bash title="Terminal"
# Looge uus Languine projekt
languine init

# See teeb:
# - Loob languine.config.js faili
# - Seab üles teie projekti struktuuri
# - Algatab vajalikud kaustad
```

### status

```bash title="Terminal"
# Kontrollige oma tõlgete staatust
languine status

# Näitab:
# - Praegune tõlke edenemine
# - Puuduvad tõlked
# - Tõlke statistika
```

### validate

```bash title="Terminal"
# Kinnitage oma tõlke failid
languine validate

# Kontrollib:
# - Faili struktuur
# - Puuduvad võtmed
# - Formaadi järjepidevus
```

### export

```bash title="Terminal"
# Ekspordige oma tõlked
languine export

# Valikud:
#   --format          Ekspordi formaat (json, yaml jne)
#   --output         Väljundi kaust
```

### import

```bash title="Terminal"
# Impordige tõlked välistest failidest
languine import

# Valikud:
#   --source         Allika fail või kaust
#   --format         Impordi formaat (json, yaml jne)
```

## Üksikud Valikud

Kõik käskud toetavad järgmisi globaalsete valikute:

```bash title="Terminal"
# Globaalsete valikute olemasolu kõigile käskudele
languine [käsk] [valikud]

# Ühised valikud:
#   --help, -h       Näita abi teavet
#   --version, -v    Näita versiooni teavet
#   --verbose        Luba üksikasjalik logimine
#   --config         Määrake konfiguratsiooni faili tee
```
