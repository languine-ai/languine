# Markdown

Tõlgitud markdown-faili käsitletakse dokumendina ja see tõlgitakse tervikuna, et säilitada dokumendi kontekst.

## Sisu ignoreerimine

Sa saad ignoreerida sisu, mis asub `<!-- languine-ignore-start -->` ja `<!-- languine-ignore-end -->` märkide vahel.

```mdx
<!-- languine-ignore-start -->
See sisu ei tõlgita.
<!-- languine-ignore-end -->
```
