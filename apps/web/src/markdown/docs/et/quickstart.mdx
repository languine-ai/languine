# Kiire algus

Alustage Languine'i installimisega.

Seejärel käivitage järgmine käsk Languine'i initsialiseerimiseks.

```bash title="Terminal"
npx languine@latest init
```

Järgige samme oma projekti seadistamiseks.

## Esimene tõlge

Nüüd olete valmis oma koodi tõlkima.

Alustage, lisades oma lähtefaili võtme, näiteks kui valite `en.json` oma lähtefailiks, lisage sellele võtme.

```json title="src/locales/en.json"
{
  "welcome": "Tere tulemast meie rakendusse!",
  "user": {
    "greeting": "Tere, {{name}}!",
    "profile": {
      "title": "Teie profiil",
      "edit": "Muuda profiili"
    }
  },
  "notifications": {
    "messages": "Teil on {{count}} uut sõnumit.",
    "empty": "Uute teatiste puudumine."
  }
}

```


## Tõlgi

Käivitage järgmine käsk oma sisu tõlkimiseks:

```bash title="Terminal"
npx languine@latest translate
```

## See teeb:
1. Loob tõlkefailid iga sihtkeele jaoks
2. Genereerib kvaliteetseid tõlkeid Languine'i mootori abil
3. Loob `languine.lock` faili tõlke olekute jälgimiseks

Lukustusfail aitab Languine'il tuvastada, millised tõlked vajavad uuendamist, kui teie lähte sisu muutub. Veenduge, et see fail oleks teie hoidlas.

## Järgmised sammud

Nüüd, kui olete Languine'i seadistanud, saate:
- Lisada oma `languine.json` faili rohkem keeli
- Integreerida Languine oma CI/CD torustikku
- Kasutada meie API-d programmiliste tõlgete jaoks

Vaadake meie teisi juhendeid, et rohkem teada saada Languine'i edasistest funktsioonidest.
