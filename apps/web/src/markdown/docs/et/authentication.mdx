# Autentimine

Languine'i kasutamiseks peate autentima meie mootoriga, see võtab vaid paar sekundit, et alustada.

---

## Sisselogimine

Saate sisse logida otse [Languine'i juhtpaneelil](https://languine.ai/login) või käivitada järgmise käsu:

```bash title="Terminal"
npx languine@latest auth login
```


Kui teil on juba konto, logitakse teid automaatselt sisse ja seanss salvestatakse teie masinasse.

---

## Väljalogimine

Saate välja logida, käivitades järgmise käsu:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Kasutaja seanss

Saate kontrollida, kas olete sisse logitud, käivitades järgmise käsu:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Keskkonnamuutujad

Saate seada järgmised keskkonnamuutujad, et konfigureerida Languine'i:

- `LANGUINE_API_KEY`: API võti, mida kasutada autentimiseks.
- `LANGUINE_PROJECT_ID`: Projekti ID, mida kasutada projekti jaoks.

Leiate need [Languine'i juhtpaneelilt](https://languine.ai/login).
