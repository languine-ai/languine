# Markdown

Tulkojot markdown failu, tas tiek uzskatīts par dokumentu un tiek tulkots kopumā, lai saglabātu dokumenta kontekstu.

## Satura ignorēšana

Jūs varat ignorēt saturu starp `<!-- languine-ignore-start -->` un `<!-- languine-ignore-end -->` tagiem.

```mdx
<!-- languine-ignore-start -->
Šis saturs netiks tulkots.
<!-- languine-ignore-end -->
```
