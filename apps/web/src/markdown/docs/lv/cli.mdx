# CLI

Languine nodrošina CLI, lai palīdzētu jums tulkot jūsu lietojumprogrammu.

## Komandu pārskats

### translate

`translate` komanda tiek izmantota, lai tulkotu jūsu lietojumprogrammu.

```bash title="Terminal"
# Sākt jūsu lietojumprogrammas tulkošanu
languine translate

# Opcijas:
#   --source-locale    Norādiet avota valodu (noklusējums: en)
#   --target-locale    Norādiet mērķa valodu(s), uz kuru tulkot
#   --config          Ceļš uz jūsu languine.config.js failu
```

### init

```bash title="Terminal"
# Inicializēt jaunu Languine projektu
languine init

# Tas izdarīs:
# - Izveidos languine.config.js failu
# - Iestatīs jūsu projekta struktūru
# - Inicializēs nepieciešamās direktorijas
```

### status

```bash title="Terminal"
# Pārbaudīt jūsu tulkojumu statusu
languine status

# Rāda:
# - Pašreizējo tulkojumu progresu
# - Trūkstošos tulkojumus
# - Tulkojumu statistiku
```

### validate

```bash title="Terminal"
# Validēt jūsu tulkojumu failus
languine validate

# Pārbauda:
# - Failu struktūru
# - Trūkstošās atslēgas
# - Formāta konsekvenci
```

### export

```bash title="Terminal"
# Eksportēt jūsu tulkojumus
languine export

# Opcijas:
#   --format          Eksporta formāts (json, yaml utt.)
#   --output         Izvades direktorija
```

### import

```bash title="Terminal"
# Importēt tulkojumus no ārējiem failiem
languine import

# Opcijas:
#   --source         Avota fails vai direktorija
#   --format         Importa formāts (json, yaml utt.)
```

## Globālās opcijas

Visas komandas atbalsta šādas globālās opcijas:

```bash title="Terminal"
# Globālās opcijas, kas pieejamas visām komandām
languine [command] [options]

# Bieži lietotās opcijas:
#   --help, -h       Rādīt palīdzības informāciju
#   --version, -v    Rādīt versijas informāciju
#   --verbose        Iespējot detalizētu žurnālfailu
#   --config         Norādiet konfigurācijas faila ceļu
```
