# CLI

Languine menyediakan CLI untuk membantu Anda menerjemahkan aplikasi Anda.

## Ikhtisar Perintah

### translate

Perintah `translate` digunakan untuk menerjemahkan aplikasi Anda.

```bash title="Terminal"
# Mulai penerjemahan aplikasi Anda
languine translate

# Opsi:
#   --source-locale    Tentukan lokal sumber (default: en)
#   --target-locale    Tentukan lokal target untuk diterjemahkan
#   --config          Jalur ke file languine.config.js Anda
```

### init

```bash title="Terminal"
# Inisialisasi proyek Languine baru
languine init

# Ini akan:
# - Membuat file languine.config.js
# - Menyiapkan struktur proyek Anda
# - Menginisialisasi direktori yang diperlukan
```

### status

```bash title="Terminal"
# Periksa status penerjemahan Anda
languine status

# Menampilkan:
# - Kemajuan penerjemahan saat ini
# - Penerjemahan yang hilang
# - Statistik penerjemahan
```

### validate

```bash title="Terminal"
# Validasi file penerjemahan Anda
languine validate

# Memeriksa:
# - Struktur file
# - Kunci yang hilang
# - Konsistensi format
```

### export

```bash title="Terminal"
# Ekspor penerjemahan Anda
languine export

# Opsi:
#   --format          Format ekspor (json, yaml, dll.)
#   --output         Direktori output
```

### import

```bash title="Terminal"
# Impor penerjemahan dari file eksternal
languine import

# Opsi:
#   --source         File atau direktori sumber
#   --format         Format impor (json, yaml, dll.)
```

## Opsi Global

Semua perintah mendukung opsi global berikut:

```bash title="Terminal"
# Opsi global tersedia untuk semua perintah
languine [command] [options]

# Opsi umum:
#   --help, -h       Tampilkan informasi bantuan
#   --version, -v    Tampilkan informasi versi
#   --verbose        Aktifkan logging verbose
#   --config         Tentukan jalur file konfigurasi
```
