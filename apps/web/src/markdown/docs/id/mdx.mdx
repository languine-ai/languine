# Markdown

File markdown yang diterjemahkan diperlakukan sebagai Dokumen, dan diterjemahkan secara keseluruhan.

## Mengabaikan konten

Anda dapat mengabaikan konten antara tag `<!-- languine-ignore-start -->` dan `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Konten ini tidak akan diterjemahkan.
<!-- languine-ignore-end -->
```
