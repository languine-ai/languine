# Autentifikacija

Da biste koristili Languine, potrebno je da se autentifikujete sa našim Engine-om, što traje samo nekoliko sekundi da biste započeli.

---

## Prijavite se

Možete se prijaviti direktno na [Languine Dashboard](https://languine.ai/login) ili pokrenuti sledeću komandu:

```bash title="Terminal"
npx languine@latest auth login
```


Ako već imate nalog, automatski ćete se prijaviti i sesija će biti sačuvana na vašem računaru.

---

## Odjavite se

Možete se odjaviti pokretanjem sledeće komande:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Korisnička sesija

Možete proveriti da li ste prijavljeni pokretanjem sledeće komande:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Promenljive okruženja

Možete postaviti sledeće promenljive okruženja da konfigurišete Languine:

- `LANGUINE_API_KEY`: API ključ koji se koristi za autentifikaciju.
- `LANGUINE_PROJECT_ID`: ID projekta koji se koristi za projekat.

Možete ih pronaći na [Languine Dashboard](https://languine.ai/login).
