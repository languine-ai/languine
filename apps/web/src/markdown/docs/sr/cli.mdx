# CLI

Languine pruža CLI alat za pomoć u prevođenju vaše aplikacije.

## Pregled komandi

### translate

Komanda `translate` se koristi za prevođenje vaše aplikacije.

```bash title="Terminal"
# Pokrenite prevođenje vaše aplikacije
languine translate

# Opcije:
#   --source-locale    Odredite izvorni jezik (podrazumevano: en)
#   --target-locale    Odredite ciljni jezik(e) za prevođenje
#   --config          Putanja do vašeg languine.config.js fajla
```

### init

```bash title="Terminal"
# Inicijalizujte novi Languine projekat
languine init

# Ovo će:
# - Kreirati languine.config.js fajl
# - Postaviti strukturu vašeg projekta
# - Inicijalizovati potrebne direktorijume
```

### status

```bash title="Terminal"
# Proverite status vaših prevoda
languine status

# Prikazuje:
# - Trenutni napredak prevođenja
# - Nedostajući prevodi
# - Statistiku prevođenja
```

### validate

```bash title="Terminal"
# Validirajte vaše fajlove sa prevodima
languine validate

# Proverava:
# - Strukturu fajla
# - Nedostajuće ključeve
# - Doslednost formata
```

### export

```bash title="Terminal"
# Izvezite vaše prevode
languine export

# Opcije:
#   --format          Format izvoza (json, yaml, itd.)
#   --output         Izlazni direktorijum
```

### import

```bash title="Terminal"
# Uvezite prevode iz spoljašnjih fajlova
languine import

# Opcije:
#   --source         Izvorni fajl ili direktorijum
#   --format         Format uvoza (json, yaml, itd.)
```

## Globalne opcije

Sve komande podržavaju sledeće globalne opcije:

```bash title="Terminal"
# Globalne opcije dostupne za sve komande
languine [command] [options]

# Uobičajene opcije:
#   --help, -h       Prikaži informacije o pomoći
#   --version, -v    Prikaži informacije o verziji
#   --verbose        Omogućite detaljno logovanje
#   --config         Odredite putanju do konfiguracionog fajla
```
