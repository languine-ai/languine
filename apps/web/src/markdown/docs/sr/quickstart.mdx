# Brzi početak

Započnite instalacijom Languine-a.

Zatim, pokrenite sledeću komandu da inicijalizujete Languine.

```bash title="Terminal"
npx languine@latest init
```

Pratite korake da postavite svoj projekat.

## Prvi prevod

Sada ste spremni da prevedete svoj kod.

Započnite dodavanjem ključa u svoj izvorni fajl, na primer, ako odlučite da koristite `en.json` kao svoj izvorni fajl, dodajte ključ u njega.

```json title="src/locales/en.json"
{
  "welcome": "Dobrodošli u našu aplikaciju!",
  "user": {
    "greeting": "Zdravo, {{name}}!",
    "profile": {
      "title": "Vaš profil",
      "edit": "Izmeni profil"
    }
  },
  "notifications": {
    "messages": "Imate {{count}} novu poruku/e.",
    "empty": "Nema novih obaveštenja."
  }
}

```


## Prevedi

Pokrenite sledeću komandu da prevedete svoj sadržaj:

```bash title="Terminal"
npx languine@latest translate
```

## Ovo će:
1. Kreirati prevodilačke fajlove za svaki ciljni jezik
2. Generisati visokokvalitetne prevode Languine Engine-a
3. Kreirati `languine.lock` fajl za praćenje stanja prevoda

Lock fajl pomaže Languine-u da identifikuje koji prevodi trebaju ažuriranje kada se vaš izvorni sadržaj promeni. Uverite se da ste ovaj fajl dodali u svoj repozitorijum.

## Sledeći koraci

Sada kada ste postavili Languine, možete:
- Dodati više jezika u svoj `languine.json` fajl
- Integrisati Languine u svoj CI/CD proces
- Koristiti naš API za programatske prevode

Pogledajte naše druge vodiče da saznate više o naprednim funkcijama Languine-a.
