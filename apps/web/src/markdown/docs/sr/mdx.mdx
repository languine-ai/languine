# Markdown

Prevedena markdown datoteka se tretira kao dokument i prevodi se u celini kako bi se sačuvao kontekst dokumenta.

## Ignorisanje sadržaja

Možete ignorisati sadržaj između `<!-- languine-ignore-start -->` i `<!-- languine-ignore-end -->` oznaka.

```mdx
<!-- languine-ignore-start -->
Ovaj sadržaj neće biti preveden.
<!-- languine-ignore-end -->
```
