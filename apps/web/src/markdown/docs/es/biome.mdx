# Bioma

Languine admite el formato con [Biome](https://biomejs.dev/) a través de hooks. Puedes configurar Biome para que formatee automáticamente tus archivos de traducción después de que se generen.

Aquí tienes un ejemplo de configuración:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Formatear archivos generados con Biome
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
