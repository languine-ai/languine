# Markdown

Un archivo markdown traducido se trata como un Documento y se traduce en su totalidad para mantener el contexto del documento.

## Ignorar contenido

Puedes ignorar el contenido entre las etiquetas `<!-- languine-ignore-start -->` y `<!-- languine-ignore-end -->`.

```mdx

<!-- languine-ignore-end -->
```
