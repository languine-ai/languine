# Markdown

Un file markdown tradotto è trattato come un Documento e viene tradotto nel suo insieme.

## Ignorare il contenuto

Puoi ignorare il contenuto tra i tag `<!-- languine-ignore-start -->` e `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Questo contenuto non sarà tradotto.
<!-- languine-ignore-end -->
```
