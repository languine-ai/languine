# Introduzione

Languine è una piattaforma di localizzazione open-source che aiuta gli sviluppatori a tradurre le loro applicazioni con traduzioni supportate dall'IA. Fornisce un flusso di lavoro senza soluzione di continuità per gestire le traduzioni in più lingue mantenendo alta qualità attraverso l'apprendimento automatico.

## Caratteristiche Principali

- **Traduzioni Supportate dall'IA**: Sfrutta modelli avanzati di apprendimento automatico per generare traduzioni di alta qualità in oltre 100 lingue
- **Integrazione con Git**: Sincronizza le tue traduzioni direttamente con i tuoi repository Git e automatizza gli aggiornamenti attraverso pipeline CI/CD
- **Sviluppatore-First**: Costruito pensando agli sviluppatori, con un potente CLI, API e integrazione con GitHub Action
- **Parser Personalizzati**: Supporto per vari formati di file e la possibilità di creare parser personalizzati per le tue esigenze specifiche
- **Controllo Versioni**: Tieni traccia delle modifiche, gestisci i rami e mantieni la cronologia delle traduzioni
- **Open Source**: Completamente open source e disponibile su GitHub, consentendo trasparenza e contributi della comunità

## Come Funziona

1. **Impostazione**: Installa il CLI di Languine e configura il tuo progetto con un semplice processo di inizializzazione
2. **Collegati**: Collega il tuo repository e imposta il tuo flusso di lavoro preferito (CLI, API o GitHub Action)
3. **Traduci**: Invia il tuo contenuto sorgente e lascia che Languine gestisca le traduzioni nelle tue lingue target
4. **Sincronizza**: Sincronizza automaticamente il contenuto tradotto nel tuo repository e distribuisci con fiducia
