# Bioma

Languine supporta la formattazione con [Biome](https://biomejs.dev/) tramite hook. Puoi configurare Biome per formattare automaticamente i tuoi file di traduzione dopo che sono stati generati.

Ecco un esempio di configurazione:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Formatta i file generati con Biome
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
