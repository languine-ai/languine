# CLI

Languine fornisce un'interfaccia a riga di comando (CLI) per aiutarti a tradurre la tua applicazione.

---

## Panoramica dei Comandi

### Traduci

Il comando `translate` viene utilizzato per tradurre la tua applicazione.

```bash title="Terminal"
    npx languine@latest translate
```

---

### Inizializza

Il comando `init` viene utilizzato per inizializzare un nuovo progetto Languine.

```bash title="Terminal"
    npx languine@latest init
```

---

### Autenticazione

Il comando `auth` viene utilizzato per autenticarsi con Languine.

```bash title="Terminal"
    npx languine@latest auth
```

---

### Sincronizza

Il comando `sync` viene utilizzato per sincronizzare la tua applicazione con Languine.

```bash title="Terminal"
    npx languine@latest sync
```

---

### Aggiungi Locale

Il comando `locale` viene utilizzato per gestire i locali della tua applicazione.

```bash title="Terminal"
    npx languine@latest locale add pt,fr
```

---

### Rimuovi Locale

Il comando `locale remove` viene utilizzato per rimuovere un locale dalla tua applicazione.

```bash title="Terminal"
    npx languine@latest locale remove pt,fr
```
