# Prettier

Languine støtter formatering med [Prettier](https://prettier.io/) gjennom hooks. Du kan konfigurere Prettier til å automatisk formatere oversettelsesfilene dine etter at de er generert.

Her er et eksempel på konfigurasjon:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    afterTranslate: async (files) => {
      await prettier.format(files, { parser: "json" });
    },
  },
});
```
