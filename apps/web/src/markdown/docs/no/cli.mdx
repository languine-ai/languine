# CLI

Languine tilbyr en CLI for å hjelpe deg med å oversette applikasjonen din.

## Kommandoversikt

### oversett

`oversett`-kommandoen brukes til å oversette applikasjonen din.

```bash title="Terminal"
# Start oversettelsen av applikasjonen din
languine oversett

# Alternativer:
#   --source-locale    Angi kilde-lokalisering (standard: en)
#   --target-locale    Angi mål-lokalisering(er) å oversette til
#   --config          Sti til din languine.config.js-fil
```

### init

```bash title="Terminal"
# Initialiser et nytt Languine-prosjekt
languine init

# Dette vil:
# - Opprette en languine.config.js-fil
# - Sette opp prosjektstrukturen din
# - Initialisere nødvendige kataloger
```

### status

```bash title="Terminal"
# Sjekk statusen for oversettelsene dine
languine status

# Viser:
# - Nåværende oversettelsesprogresjon
# - Manglende oversettelser
# - Oversettelsesstatistikk
```

### valider

```bash title="Terminal"
# Valider oversettelsesfilene dine
languine valider

# Sjekker:
# - Filstruktur
# - Manglende nøkler
# - Formatkonsistens
```

### eksporter

```bash title="Terminal"
# Eksporter oversettelsene dine
languine eksporter

# Alternativer:
#   --format          Eksporteringsformat (json, yaml, osv.)
#   --output         Utdata-katalog
```

### importer

```bash title="Terminal"
# Importer oversettelser fra eksterne filer
languine importer

# Alternativer:
#   --source         Kildefil eller katalog
#   --format         Importeringsformat (json, yaml, osv.)
```

## Globale alternativer

Alle kommandoer støtter følgende globale alternativer:

```bash title="Terminal"
# Globale alternativer tilgjengelig for alle kommandoer
languine [kommando] [alternativer]

# Vanlige alternativer:
#   --help, -h       Vis hjelpeinformasjon
#   --version, -v    Vis versjonsinformasjon
#   --verbose        Aktiver detaljert logging
#   --config         Angi sti til konfigurasjonsfil
```
