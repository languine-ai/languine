# Introduksjon

Languine er en åpen kildekode lokaliseringsplattform som hjelper utviklere med å oversette applikasjonene sine med AI-drevne oversettelser. Den gir en sømløs arbeidsflyt for å administrere oversettelser på tvers av flere språk, samtidig som den opprettholder høy kvalitet gjennom maskinlæring.

## Nøkkelfunksjoner

- **AI-Drevne Oversettelser**: Utnytt avanserte maskinlæringsmodeller for å generere høykvalitets oversettelser på over 100 språk
- **Git Integrasjon**: Synkroniser oversettelsene dine direkte med Git-repositoriene dine og automatiser oppdateringer gjennom CI/CD-pipelines
- **Utvikler-Først**: Bygget med utviklere i tankene, med en kraftig CLI, API og GitHub Action-integrasjon
- **Egendefinerte Parser**: Støtte for ulike filformater og muligheten til å lage egendefinerte parser for dine spesifikke behov
- **Versjonskontroll**: Spor endringer, administrer grener og oppretthold oversettelseshistorikk
- **Åpen Kildekode**: Fullt åpen kildekode og tilgjengelig på GitHub, som muliggjør åpenhet og bidrag fra fellesskapet

## Hvordan Det Fungerer

1. **Oppsett**: Installer Languine CLI og konfigurer prosjektet ditt med en enkel initieringsprosess
2. **Koble til**: Koble til repositoriet ditt og sett opp din foretrukne arbeidsflyt (CLI, API eller GitHub Action)
3. **Oversett**: Push innholdet ditt og la Languine håndtere oversettelsene til målspråkene dine
4. **Synkroniser**: Automatisk synkronisere oversatt innhold tilbake til repositoriet ditt og distribuer med tillit
