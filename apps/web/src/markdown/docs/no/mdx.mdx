# Markdown

En oversatt markdown-fil behandles som et dokument, og oversettes som en helhet.

## Ignorer innhold

Du kan ignorere innhold mellom `<!-- languine-ignore-start -->` og `<!-- languine-ignore-end -->`-tagger.

```mdx
<!-- languine-ignore-start -->
Dette innholdet vil ikke bli oversatt.
<!-- languine-ignore-end -->
```
