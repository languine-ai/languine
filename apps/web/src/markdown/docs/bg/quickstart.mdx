# Бързо начало

Започнете с инсталирането на Languine.

След това изпълнете следната команда, за да инициализирате Languine.

```bash title="Терминал"
npx languine@latest init
```

Следвайте стъпките, за да настроите проекта си.

## Първият превод

Сега сте готови да преведете кода си.

Започнете, като добавите ключ към изходния си файл, например, ако решите да използвате `en.json` като изходен файл, добавете ключ към него.

```json title="src/locales/en.json"
{
  "welcome": "Добре дошли в нашето приложение!",
  "user": {
    "greeting": "Здравей, {{name}}!",
    "profile": {
      "title": "Вашият профил",
      "edit": "Редактиране на профила"
    }
  },
  "notifications": {
    "messages": "Имате {{count}} ново съобщение(я).",
    "empty": "Няма нови известия."
  }
}

```


## Превод

Изпълнете следната команда, за да преведете съдържанието си:

```bash title="Терминал"
npx languine@latest translate
```

## Това ще:
1. Създаде файлове за превод за всеки целеви език
2. Генерира висококачествени преводи с Languine Engine
3. Създаде файл `languine.lock`, за да проследява състоянието на преводите

Файлът за заключване помага на Languine да идентифицира кои преводи се нуждаят от актуализации, когато изходното съдържание се променя. Уверете се, че сте ангажирали този файл в репозитория си.

## Следващи стъпки

Сега, когато имате настроен Languine, можете:
- Да добавите още езици във файла `languine.json`
- Да интегрирате Languine в CI/CD процеса си
- Да използвате нашето API за програмни преводи

Разгледайте и другите ни ръководства, за да научите повече за напредналите функции на Languine.
