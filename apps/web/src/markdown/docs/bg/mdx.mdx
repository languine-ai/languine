# Маркдаун

Преведеният файл с маркдаун се третира като документ и се превежда като цяло, за да се запази контекстът на документа.

## Игнориране на съдържание

Можете да игнорирате съдържание между `<!-- languine-ignore-start -->` и `<!-- languine-ignore-end -->` тагове.

```mdx
<!-- languine-ignore-start -->
Това съдържание няма да бъде преведено.
<!-- languine-ignore-end -->
```
