# JSON

Languine поддържа JSON файлове като изходен формат с общите изисквания за множествено число и интерполация. Този формат се използва в много рамки като React, Vue, Ruby on Rails и други.


---


## Настройка

Първо, уверете се, че имате languine.json или languine.ts конфигурационен файл в корена на проекта си. Ето един пример:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Превод

С конфигурацията настроена, стартирайте:

```bash
npx languine@latest translate
```

Когато стартирате тази команда, Languine ще:

- Зареди вашия изходен JSON файл (напр. locales/en.json)
- Открие нови или променени стрингове за превод
- Генерира преводи за целевите ви езици
- Създаде или актуализира JSON файловете за целевия език (напр. locales/sv.json, locales/de.json, locales/fr.json)
