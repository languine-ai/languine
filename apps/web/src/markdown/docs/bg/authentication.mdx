# Аутентикация

За да използвате Languine, трябва да се аутентикирате с нашия Engine, това отнема само няколко секунди, за да започнете.

---

## Вход

Можете да влезете директно в [Languine Dashboard](https://languine.ai/login) или да изпълните следната команда:

```bash title="Terminal"
npx languine@latest auth login
```


Ако вече имате акаунт, ще бъдете автоматично влезли и сесия ще бъде запазена на вашата машина.

---

## Изход

Можете да излезете, като изпълните следната команда:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Потребителска сесия

Можете да проверите дали сте влезли, като изпълните следната команда:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Променливи на средата

Можете да зададете следните променливи на средата, за да конфигурирате Languine:

- `LANGUINE_API_KEY`: API ключът, който да използвате за аутентикация.
- `LANGUINE_PROJECT_ID`: Идентификаторът на проекта, който да използвате за проекта.

Можете да ги намерите в [Languine Dashboard](https://languine.ai/login).
