# Hitelesítés

A Languine használatához hitelesítened kell magad a Motorunkkal, ez csak néhány másodpercet vesz igénybe a kezdéshez.

---

## Bejelentkezés

Közvetlenül bejelentkezhetsz a [Languine Dashboard](https://languine.ai/login) oldalán, vagy futtathatod a következő parancsot:

```bash title="Terminál"
npx languine@latest auth login
```


Ha már van fiókod, automatikusan be leszel jelentkezve, és egy munkamenet mentésre kerül a gépeden.

---

## Kijelentkezés

A következő parancs futtatásával kijelentkezhetsz:

```bash title="Terminál"
npx languine@latest auth logout
```

---

## Felhasználói munkamenet

Ellenőrizheted, hogy be vagy-e jelentkezve a következő parancs futtatásával:

```bash title="Terminál"
npx languine@latest auth whoami
```

---


# Környezeti változók

A következő környezeti változókat állíthatod be a Languine konfigurálásához:

- `LANGUINE_API_KEY`: Az API kulcs, amelyet a hitelesítéshez használsz.
- `LANGUINE_PROJECT_ID`: A projekt azonosító, amelyet a projekthez használsz.

Ezeket a [Languine Dashboard](https://languine.ai/login) oldalon találod.
