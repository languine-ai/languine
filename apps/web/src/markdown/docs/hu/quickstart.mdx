# Gyors kezdés

Kezdje azzal, hogy telepíti a Languine-t.

Ezután futtassa az alábbi parancsot a Languine inicializálásához.

```bash title="Terminál"
npx languine@latest init
```

Kövesse a lépéseket a projekt beállításához.

## Első fordítás

Most készen áll a kód fordítására.

Kezdje azzal, hogy hozzáad egy kulcsot a forrásfájlhoz, például ha az `en.json`-t választja forrásfájlul, adjon hozzá egy kulcsot.

```json title="src/locales/en.json"
{
  "welcome": "Üdvözöljük alkalmazásunkban!",
  "user": {
    "greeting": "Szia, {{name}}!",
    "profile": {
      "title": "A profilod",
      "edit": "Profil szerkesztése"
    }
  },
  "notifications": {
    "messages": "{{count}} új üzenete van.",
    "empty": "Nincsenek új értesítések."
  }
}

```


## Fordítás

Futtassa az alábbi parancsot a tartalom fordításához:

```bash title="Terminál"
npx languine@latest translate
```

## Ez a következőket fogja tenni:
1. Fordítási fájlokat hoz létre minden célnyelvhez
2. Magas színvonalú fordításokat generál a Languine Engine segítségével
3. Létrehoz egy `languine.lock` fájlt a fordítási állapotok nyomon követésére

A zárolási fájl segít a Languine-nak azonosítani, hogy mely fordítások igényelnek frissítést, amikor a forrás tartalom megváltozik. Győződjön meg róla, hogy ezt a fájlt elkötelezi a tárolójában.

## Következő lépések

Most, hogy beállította a Languine-t, lehetősége van:
- További nyelveket hozzáadni a `languine.json` fájlhoz
- Integrálni a Languine-t a CI/CD folyamatába
- Használni az API-nkat programozott fordításokhoz

Nézze meg más útmutatóinkat, hogy többet megtudjon a Languine fejlett funkcióiról.
