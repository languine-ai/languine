# Markdown

A lefordított markdown fájl Dokumentumként van kezelve, és egészében van lefordítva.

## Tartalom figyelmen kívül hagyása

A `<!-- languine-ignore-start -->` és `<!-- languine-ignore-end -->` tagek közötti tartalmat figyelmen kívül hagyhatja.

```mdx
<!-- languine-ignore-start -->
Ez a tartalom nem lesz lefordítva.
<!-- languine-ignore-end -->
```
