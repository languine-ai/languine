# Markdown

A lefordított markdown fájl Dokumentumként van kezelve, és az egész fájlt lefordítják, hogy megőrizzék a dokumentum kontextusát.

## Tartalom figyelmen kívül hagyása

A `<!-- languine-ignore-start -->` és `<!-- languine-ignore-end -->` címkék közötti tartalmat figyelmen kívül hagyhatja.

```mdx
<!-- languine-ignore-start -->
Ez a tartalom nem lesz lefordítva.
<!-- languine-ignore-end -->
```
