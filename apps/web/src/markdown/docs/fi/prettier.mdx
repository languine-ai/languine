# Prettier

Languine tukee muotoilua [Prettierin](https://prettier.io/) kautta koukkujen avulla. Voit määrittää Prettierin automaattisesti muotoilemaan käännöstiedostosi niiden luomisen jälkeen.

Tässä on esimerkkikonfiguraatio:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    afterTranslate: async (files) => {
      await prettier.format(files, { parser: "json" });
    },
  },
});
```
