## Avainominaisuudet

- **Tekoälypohjaiset käännökset**: Hyödynnä edistyneitä koneoppimismalleja tuottamaan korkealaatuisia käännöksiä yli 100 kielellä
- **Git-integraatio**: Synkronoi käännöksesi suoraan Git-repositorioidesi kanssa ja automatisoi päivitykset CI/CD-putkien kautta
- **Kehittäjäystävällinen**: Rakennettu kehittäjät mielessä, sisältäen tehokkaan CLI:n, API:n ja GitHub Action -integraation
- **Mukautetut jäsentäjät**: Tuki erilaisille tiedostomuodoille ja mahdollisuus luoda mukautettuja jäsentäjiä erityistarpeitasi varten
- **Versionhallinta**: Seuraa muutoksia, hallitse haaroja ja ylläpidä käännöshistoriaa
- **Avoin lähdekoodi**: Täysin avoin lähdekoodi ja saatavilla GitHubissa, mikä mahdollistaa läpinäkyvyyden ja yhteisön panostukset

## Kuinka se toimii

1. **Asennus**: Asenna Languine CLI ja määritä projektisi yksinkertaisella aloitusprosessilla
2. **Yhdistä**: Liitä repositoriosi ja määritä haluamasi työnkulku (CLI, API tai GitHub Action)
3. **Käännä**: Puske lähdesisältösi ja anna Languinen hoitaa käännökset kohdekielillesi
4. **Synkronoi**: Synkronoi automaattisesti käännetty sisältö takaisin repositoriosi ja julkaise luottavaisin mielin

## Aloittaminen

Aloittaaksesi Languinen käytön, seuraa [Asennus](/docs/installation) opasta asentaaksesi CLI:n ja luodaksesi ensimmäisen projektisi. Tarkista sitten [Nopea aloitus](/docs/quickstart) opas oppiaksesi käännösten hallinnan perusteet Languinella.

Yksityiskohtaisempaa tietoa erityisominaisuuksista ja edistyneestä käytöstä löydät kattavista dokumentaatio-osioistamme:

- [Määritykset](/docs/configuration) - Opi, kuinka määrittää Languine projektiisi
- [CLI](/docs/cli) - Tutustu CLI-komentojen koko valikoimaan
- [API](/docs/api) - Integroi Languine ohjelmallisesti
- [Työnkulut](/docs/workflows) - Määritä automatisoidut käännöstyönkulut
