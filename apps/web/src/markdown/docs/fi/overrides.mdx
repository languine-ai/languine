# Ohitukset

Languine antaa sinun ohittaa käännökset tietyille avaimille. Tämä on hyödyllistä, kun haluat varmistaa, että tietty käännös käytetään tietylle avaimelle riippumatta oletuskäännöksestä.

## Kuinka se toimii

Ohjauspaneelissa voit luoda ohituksia tietyille avaimille. Voit myös luoda ohituksia tietyille kielille.
