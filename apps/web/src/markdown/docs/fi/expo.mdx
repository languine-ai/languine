# Expo

Languine tukee Expo-lokalisointia, ja se integroituu saumattomasti Expon i18n-järjestelmään. Tämä muoto toimii sekä JavaScript- että TypeScript-projekteissa, ja se tukee sisäkkäisiä rakenteita ja monikkomuotoja.

---

## Asetusten määrittäminen

Ensinnäkin varmista, että sinulla on languine.config.json tai languine.config.ts -konfiguraatiotiedosto projektisi juurihakemistossa. Tässä on esimerkki:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "expo": {
      "include": ["assets/locales/[locale].json"]
    }
  }
}
```

## Kääntäminen

Kun konfiguraatiosi on asetettu, suorita:

```bash
npx languine@latest translate
```

Kun suoritat tämän komennon, Languine:

- Lataa lähdekäännöstiedostosi (esim. assets/locales/en.json)
- Havaitsee kaikki uudet tai muokatut käännösmerkkijonot
- Luo käännöksiä kohdekielillesi
- Luo tai päivittää kohdekielitiedostot (esim. assets/locales/sv.json)
- Säilyttää sisäkkäisen rakenteen ja monikkomuotojen säännöt
- Luo TypeScript-tyyppejä, jos se on käytössä
- Säilyttää yhteensopivuuden expo-localizationin kanssa
