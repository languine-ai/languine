# Markdown

Käännetty markdown-tiedosto käsitellään asiakirjana, ja se käännetään kokonaisuutena asiakirjan kontekstin säilyttämiseksi.

## Sisällön ohittaminen

Voit ohittaa sisällön `<!-- languine-ignore-start -->` ja `<!-- languine-ignore-end -->` -tagejen välissä.

```mdx
<!-- languine-ignore-start -->
Tätä sisältöä ei käännetä.
<!-- languine-ignore-end -->
```
