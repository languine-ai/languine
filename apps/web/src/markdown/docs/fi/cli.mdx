# CLI

Languine tarjoaa CLI:n, joka auttaa sinua kääntämään sovelluksesi.

## Komentojen Yleiskatsaus

### translate

`translate`-komentoa käytetään sovelluksesi kääntämiseen.

```bash title="Terminal"
# Aloita sovelluksesi kääntäminen
languine translate

# Vaihtoehdot:
#   --source-locale    Määritä lähdekieli (oletus: en)
#   --target-locale    Määritä kohdekieli(t), joihin käännetään
#   --config          Polku languine.config.js-tiedostollesi
```

### init

```bash title="Terminal"
# Alusta uusi Languine-projekti
languine init

# Tämä tekee:
# - Luo languine.config.js-tiedoston
# - Asettaa projektirakenteesi
# - Alustaa tarvittavat hakemistot
```

### status

```bash title="Terminal"
# Tarkista käännöksiesi tila
languine status

# Näyttää:
# - Nykyinen käännöksen edistyminen
# - Puuttuvat käännökset
# - Käännösstatistiikka
```

### validate

```bash title="Terminal"
# Vahvista käännöstiedostosi
languine validate

# Tarkistaa:
# - Tiedostorakenne
# - Puuttuvat avaimet
# - Muotojohdonmukaisuus
```

### export

```bash title="Terminal"
# Vie käännöksesi
languine export

# Vaihtoehdot:
#   --format          Vientimuoto (json, yaml, jne.)
#   --output         Tulostushakemisto
```

### import

```bash title="Terminal"
# Tuo käännöksiä ulkoisista tiedostoista
languine import

# Vaihtoehdot:
#   --source         Lähdetiedosto tai hakemisto
#   --format         Tuo muoto (json, yaml, jne.)
```

## Globaalit Vaihtoehdot

Kaikki komennot tukevat seuraavia globaaleja vaihtoehtoja:

```bash title="Terminal"
# Globaalit vaihtoehdot kaikille komennoille
languine [komento] [vaihtoehdot]

# Yleiset vaihtoehdot:
#   --help, -h       Näytä ohjeinformaatio
#   --version, -v    Näytä versiotiedot
#   --verbose        Ota käyttöön yksityiskohtainen lokitus
#   --config         Määritä konfiguraatiotiedoston polku
```
