# Biomi

Languine tukee muotoilua [Biomilla](https://biomejs.dev/) koukkujen avulla. Voit määrittää Biomin automaattisesti muotoilemaan käännöstiedostosi niiden luomisen jälkeen.

Tässä on esimerkkikonfiguraatio:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Muotoile luodut tiedostot Biomin avulla
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
