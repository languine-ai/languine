# ARB (응용 프로그램 리소스 번들)

Languine은 Flutter 애플리케이션의 표준 지역화 형식인 ARB 파일을 지원합니다. 이 형식은 Flutter의 intl 패키지와의 호환성을 유지하면서 복수형, 자리 표시자 및 선택 표현을 처리합니다.

---

## 설정하기

먼저, 프로젝트 루트에 languine.config.json 또는 languine.config.ts 구성 파일이 있는지 확인하세요. 다음은 예시입니다:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "arb": {
      "include": ["lib/l10n/intl_[locale].arb"]
    }
  }
}
```

## 번역하기

구성이 완료되면 다음 명령어를 실행하세요:

```bash
npx languine@latest translate
```

이 명령어를 실행하면 Languine은:

- 소스 ARB 파일을 로드합니다 (예: lib/l10n/intl_en.arb)
- 새로 추가되거나 수정된 번역 문자열을 감지합니다
- 대상 언어에 대한 번역을 생성합니다
- 대상 언어 ARB 파일을 생성하거나 업데이트합니다 (예: lib/l10n/intl_sv.arb)
- 메타데이터, 자리 표시자 및 복수형/선택 표현을 보존합니다
