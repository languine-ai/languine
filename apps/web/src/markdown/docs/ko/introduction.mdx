# 소개

## 주요 기능

- **AI 기반 번역**: 100개 이상의 언어에 걸쳐 고품질 번역을 생성하기 위해 고급 기계 학습 모델을 활용합니다.
- **Git 통합**: 번역을 Git 리포지토리와 직접 동기화하고 CI/CD 파이프라인을 통해 업데이트를 자동화합니다.
- **개발자 우선**: 개발자를 염두에 두고 설계되었으며, 강력한 CLI, API 및 GitHub Action 통합 기능을 제공합니다.
- **사용자 정의 파서**: 다양한 파일 형식을 지원하며 특정 요구에 맞는 사용자 정의 파서를 생성할 수 있습니다.
- **버전 관리**: 변경 사항을 추적하고, 브랜치를 관리하며, 번역 이력을 유지합니다.
- **오픈 소스**: 완전한 오픈 소스이며 GitHub에서 사용할 수 있어 투명성과 커뮤니티 기여를 가능하게 합니다.

## 작동 방식

1. **설정**: Languine CLI를 설치하고 간단한 초기화 프로세스를 통해 프로젝트를 구성합니다.
2. **연결**: 리포지토리를 연결하고 선호하는 워크플로(CLI, API 또는 GitHub Action)를 설정합니다.
3. **번역**: 소스 콘텐츠를 푸시하고 Languine이 대상 언어로 번역을 처리하도록 합니다.
4. **동기화**: 번역된 콘텐츠를 리포지토리로 자동으로 동기화하고 자신 있게 배포합니다.

## 시작하기

Languine 사용을 시작하려면 [설치](/docs/installation) 가이드를 따라 CLI를 설정하고 첫 번째 프로젝트를 생성하세요. 그런 다음 [빠른 시작](/docs/quickstart) 가이드를 확인하여 Languine으로 번역을 관리하는 기본 사항을 배우세요.

특정 기능 및 고급 사용에 대한 자세한 정보는 포괄적인 문서 섹션을 탐색하세요:

- [구성](/docs/configuration) - 프로젝트에 맞게 Languine을 구성하는 방법을 배웁니다.
- [CLI](/docs/cli) - 전체 CLI 명령 범위를 확인합니다.
- [API](/docs/api) - Languine을 프로그래밍 방식으로 통합합니다.
- [워크플로](/docs/workflows) - 자동화된 번역 워크플로를 설정합니다.
