# 생태계

Languine은 [Biome](https://biomejs.dev/)를 통해 후크로 포맷팅을 지원합니다. 생성된 후 번역 파일을 자동으로 포맷하도록 Biome을 구성할 수 있습니다.

다음은 구성 예시입니다:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Biome으로 생성된 파일 포맷팅
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
