# 빠른 시작

Languine을 설치하는 것으로 시작하세요.

그런 다음, 다음 명령어를 실행하여 Languine을 초기화하세요.

```bash title="터미널"
npx languine@latest init
```

단계를 따라 프로젝트를 설정하세요.

---

## 첫 번째 번역

이제 코드를 번역할 준비가 되었습니다.

소스 파일에 키를 추가하는 것으로 시작하세요. 예를 들어, `en.json`을 소스 파일로 선택한 경우, 그 파일에 키를 추가하세요.

```json title="src/locales/en.json"
{
  "welcome": "우리 애플리케이션에 오신 것을 환영합니다!",
  "user": {
    "greeting": "안녕하세요, {{name}}!",
    "profile": {
      "title": "당신의 프로필",
      "edit": "프로필 수정"
    }
  },
  "notifications": {
    "messages": "새 메시지가 {{count}}개 있습니다.",
    "empty": "새 알림이 없습니다."
  }
}

```

---

## 번역

다음 명령어를 실행하여 콘텐츠를 번역하세요:

```bash title="터미널"
npx languine@latest translate
```

## 이 작업은:
1. 각 대상 언어에 대한 번역 파일을 생성합니다.
2. Languine 엔진으로 고품질 번역을 생성합니다.
3. 번역 상태를 추적하기 위해 `languine.lock` 파일을 생성합니다.

잠금 파일은 Languine이 소스 콘텐츠가 변경될 때 어떤 번역이 업데이트가 필요한지 식별하는 데 도움을 줍니다. 이 파일을 리포지토리에 커밋하는 것을 잊지 마세요.
