# Prettier

Languine은 후크를 통해 [Prettier](https://prettier.io/)로 포맷팅을 지원합니다. 생성된 후 번역 파일을 자동으로 포맷팅하도록 Prettier를 구성할 수 있습니다.

다음은 구성 예시입니다:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    afterTranslate: async (files) => {
      await prettier.format(files, { parser: "json" });
    },
  },
});
```
