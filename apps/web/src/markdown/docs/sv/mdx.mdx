# Markdown

En översatt markdown-fil behandlas som ett dokument och översätts som en helhet för att bevara dokumentets sammanhang.

## Ignorera innehåll

Du kan ignorera innehåll mellan `<!-- languine-ignore-start -->` och `<!-- languine-ignore-end -->` taggar.

```mdx
<!-- languine-ignore-start -->
Detta innehåll kommer inte att översättas.
<!-- languine-ignore-end -->
```
