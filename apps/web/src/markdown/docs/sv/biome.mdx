# Biome

Languine stöder formatering med [Biome](https://biomejs.dev/) genom hooks. Du kan konfigurera Biome för att automatiskt formatera dina översättningsfiler efter att de har genererats.

Här är ett exempel på konfiguration:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Formatera genererade filer med Biome
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
