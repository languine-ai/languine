# Introduktion

Languine är en öppen källkodsplattform för lokalisering som hjälper utvecklare att översätta sina applikationer med AI-drivna översättningar. Den erbjuder ett sömlöst arbetsflöde för att hantera översättningar över flera språk samtidigt som hög kvalitet upprätthålls genom maskininlärning.

## Nyckelfunktioner

- **AI-drivna översättningar**: Utnyttja avancerade maskininlärningsmodeller för att generera högkvalitativa översättningar på över 100 språk
- **Git-integration**: Synkronisera dina översättningar direkt med dina Git-repositorier och automatisera uppdateringar genom CI/CD-pipelines
- **Utvecklarfokus**: Byggd med utvecklare i åtanke, med en kraftfull CLI, API och GitHub Action-integration
- **Anpassade parser**: Stöd för olika filformat och möjlighet att skapa anpassade parser för dina specifika behov
- **Versionskontroll**: Spåra ändringar, hantera grenar och upprätthålla översättningshistorik
- **Öppen källkod**: Helt öppen källkod och tillgänglig på GitHub, vilket möjliggör transparens och bidrag från gemenskapen

## Hur det fungerar

1. **Installation**: Installera Languine CLI och konfigurera ditt projekt med en enkel initieringsprocess
2. **Anslut**: Koppla ditt repository och ställ in ditt föredragna arbetsflöde (CLI, API eller GitHub Action)
3. **Översätt**: Skicka ditt källinnehåll och låt Languine hantera översättningarna till dina målspråk
4. **Synkronisera**: Automatisk synkronisering av översatt innehåll tillbaka till ditt repository och distribuera med förtroende
