# Överskrivningar

Languine gör det möjligt att överskriva översättningar för specifika nycklar. Detta är användbart när du vill säkerställa att en specifik översättning används för en viss nyckel, oavsett den standardöversättning som finns.

## Hur det fungerar

I instrumentpanelen kan du skapa överskrivningar för specifika nycklar. Du kan också skapa överskrivningar för specifika språk.
