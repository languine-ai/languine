# Markdown

Languine stödjer lokalisering av Markdown-filer, perfekt för dokumentation, blogginlägg och innehållsrika webbplatser. Detta format bevarar Markdown-syntaxen medan innehållet översätts, inklusive stöd för front matter.

---

## Installera

Först, se till att du har en languine.config.json eller languine.config.ts konfigurationsfil i rotmappen av ditt projekt. Här är ett exempel:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "md": {
      "include": ["content/[locale]/**/*.md"],
      "frontMatter": true,  // Valfritt: standard är true
      "translateFrontMatterFields": ["title", "description"]  // Valfritt: specificera vilka front matter-fält som ska översättas
    }
  }
}
```

## Översättning

Med din konfiguration inställd, kör:

```bash
npx languine@latest translate
```

När du kör det här kommandot kommer Languine att:

- Ladda dina käll-Mardown-filer (t.ex. content/en/*.md)
- Upptäcka nytt eller ändrat innehåll
- Generera översättningar för dina målspråk
- Skapa eller uppdatera målspråksfiler (t.ex. content/sv/*.md)
- Bevara Markdown-format och struktur
- Hantera översättningar av front matter
- Upprätthålla länkar och referenser
- Hålla kodblock oöversatta
- Bevara HTML inom Markdown
