# JSON

Languine stöder JSON-filer som ett källformat med de vanliga kraven för pluralisering och interpolering. Detta format används i många ramverk som React, Vue, Ruby on Rails och mer.


---


## Inställning

Först, se till att du har en languine.json-konfigurationsfil i rotmappen av ditt projekt. Här är ett exempel:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Översättning

Med din konfiguration inställd, kör:

```bash
npx languine@latest translate
```

När du kör detta kommando kommer Languine att:

- Ladda din käll-JSON-fil (t.ex. locales/en.json)
- Upptäcka eventuella nya eller ändrade översättningssträngar
- Generera översättningar för dina målspråk
- Skapa eller uppdatera målspråkens JSON-filer (t.ex. locales/sv.json, locales/de.json, locales/fr.json)
