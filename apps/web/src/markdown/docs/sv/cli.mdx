# CLI

Languine tillhandahåller en CLI för att hjälpa dig att översätta din applikation.

## Kommandon Översikt

### översätt

Kommandot `översätt` används för att översätta din applikation.

```bash title="Terminal"
# Starta översättningen av din applikation
languine översätt

# Alternativ:
#   --source-locale    Ange källspråket (standard: en)
#   --target-locale    Ange målspråk för översättning
#   --config          Sökväg till din languine.config.js-fil
```

### init

```bash title="Terminal"
# Initiera ett nytt Languine-projekt
languine init

# Detta kommer att:
# - Skapa en languine.config.js-fil
# - Ställa in din projektstruktur
# - Initiera nödvändiga kataloger
```

### status

```bash title="Terminal"
# Kontrollera statusen för dina översättningar
languine status

# Visar:
# - Aktuell översättningsprogress
# - Saknade översättningar
# - Översättningsstatistik
```

### validera

```bash title="Terminal"
# Validera dina översättningsfiler
languine validera

# Kontrollerar:
# - Filstruktur
# - Saknade nycklar
# - Formatkonsekvens
```

### exportera

```bash title="Terminal"
# Exportera dina översättningar
languine exportera

# Alternativ:
#   --format          Exportformat (json, yaml, etc.)
#   --output         Utmatningskatalog
```

### importera

```bash title="Terminal"
# Importera översättningar från externa filer
languine importera

# Alternativ:
#   --source         Källfil eller katalog
#   --format         Importformat (json, yaml, etc.)
```

## Globala Alternativ

Alla kommandon stöder följande globala alternativ:

```bash title="Terminal"
# Globala alternativ tillgängliga för alla kommandon
languine [kommando] [alternativ]

# Vanliga alternativ:
#   --help, -h       Visa hjälpinformation
#   --version, -v    Visa versionsinformation
#   --verbose        Aktivera detaljerad loggning
#   --config         Ange sökväg till konfigurationsfil
```
