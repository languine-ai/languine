# Markdown

Một tệp markdown đã được dịch được coi là một Tài liệu và được dịch toàn bộ.

## Bỏ qua nội dung

Bạn có thể bỏ qua nội dung giữa các thẻ `<!-- languine-ignore-start -->` và `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Nội dung này sẽ không được dịch.
<!-- languine-ignore-end -->
```
