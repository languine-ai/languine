# Xác thực

Để sử dụng Languine, bạn cần xác thực với Engine của chúng tôi, chỉ mất vài giây để bắt đầu.

---

## Đăng nhập

Bạn có thể đăng nhập trực tiếp trên [Bảng điều khiển Languine](https://languine.ai/login) hoặc chạy lệnh sau:

```bash title="Terminal"
npx languine@latest auth login
```


Nếu bạn đã có tài khoản, bạn sẽ được đăng nhập tự động và một phiên sẽ được lưu trên máy của bạn.

---

## Đăng xuất

Bạn có thể đăng xuất bằng cách chạy lệnh sau:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Phiên người dùng

Bạn có thể kiểm tra xem bạn đã đăng nhập hay chưa bằng cách chạy lệnh sau:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Biến môi trường

Bạn có thể thiết lập các biến môi trường sau để cấu hình Languine:

- `LANGUINE_API_KEY`: Khóa API để sử dụng cho xác thực.
- `LANGUINE_PROJECT_ID`: ID dự án để sử dụng cho dự án.

Bạn có thể tìm thấy chúng trong [Bảng điều khiển Languine](https://languine.ai/login).
