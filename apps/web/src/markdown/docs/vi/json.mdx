# JSON

Languine hỗ trợ các tệp JSON như một định dạng nguồn với các yêu cầu chung về số nhiều và nội suy. Định dạng này được sử dụng trong nhiều framework như React, Vue, Ruby on Rails, và nhiều hơn nữa.


---


## Thiết Lập

Đầu tiên, hãy chắc chắn rằng bạn đã có tệp cấu hình languine.json hoặc languine.ts trong thư mục gốc của dự án. Đây là một ví dụ:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Dịch Thuật

Với cấu hình của bạn đã được thiết lập, hãy chạy:

```bash
npx languine@latest translate
```

Khi bạn chạy lệnh này, Languine sẽ:

- Tải tệp JSON nguồn của bạn (ví dụ: locales/en.json)
- Phát hiện bất kỳ chuỗi dịch thuật mới hoặc đã được sửa đổi
- Tạo ra các bản dịch cho các ngôn ngữ mục tiêu của bạn
- Tạo hoặc cập nhật các tệp JSON ngôn ngữ mục tiêu (ví dụ: locales/sv.json, locales/de.json, locales/fr.json)
