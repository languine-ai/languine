# JSON

Languine JSON fayllarını çoxluq və interpolasiya üçün ümumi tələblərlə mənbə formatı kimi dəstəkləyir. Bu format React, Vue, Ruby on Rails və daha çox çərçivələrdə istifadə olunur.


---


## Quraşdırma

İlk öncə, layihə kökünüzdə languine.json və ya languine.ts konfiqurasiya faylınızın olduğuna əmin olun. Budur bir nümunə:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Tərcümə

Konfiqurasiyanız hazır olduqda, aşağıdakı əmri icra edin:

```bash
npx languine@latest translate
```

Bu əmri icra etdikdə, Languine:

- Mənbə JSON faylınızı yükləyəcək (məsələn, locales/en.json)
- Hər hansı yeni və ya dəyişdirilmiş tərcümə sətirlərini aşkar edəcək
- Hədəf dilləriniz üçün tərcümələr yaradacaq
- Hədəf dil JSON fayllarını yaradacaq və ya yeniləyəcək (məsələn, locales/sv.json, locales/de.json, locales/fr.json)
