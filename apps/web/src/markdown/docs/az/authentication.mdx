# Təsdiq

Languine-dən istifadə etmək üçün bizim Mühərrik ilə təsdiq etməlisiniz, başlamaq üçün yalnız bir neçə saniyə çəkir.

---

## Daxil olun

Birbaşa [Languine Dashboard](https://languine.ai/login) səhifəsində daxil ola bilərsiniz və ya aşağıdakı əmri icra edə bilərsiniz:

```bash title="Terminal"
npx languine@latest auth login
```


Əgər artıq bir hesabınız varsa, avtomatik olaraq daxil olacaqsınız və sessiya kompüterinizdə saxlanılacaq.

---

## Çıxış edin

Aşağıdakı əmri icra edərək çıxış edə bilərsiniz:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## İstifadəçi sessiyası

Daxil olub-olmadığınızı yoxlamaq üçün aşağıdakı əmri icra edə bilərsiniz:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Mühit Dəyişənləri

Languine-i konfiqurasiya etmək üçün aşağıdakı mühit dəyişənlərini təyin edə bilərsiniz:

- `LANGUINE_API_KEY`: Təsdiq üçün istifadə ediləcək API açarı.
- `LANGUINE_PROJECT_ID`: Layihə üçün istifadə ediləcək layihə ID.

Onları [Languine Dashboard](https://languine.ai/login) səhifəsində tapa bilərsiniz.
