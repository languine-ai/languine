# JSON

Languineは、複数形化や補間に関する一般的な要件を持つソースフォーマットとしてJSONファイルをサポートしています。このフォーマットは、React、Vue、Ruby on Railsなどの多くのフレームワークで使用されています。


---


## セットアップ

まず、プロジェクトのルートにlanguine.jsonまたはlanguine.tsの設定ファイルがあることを確認してください。以下はその例です：

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## 翻訳

設定が完了したら、次のコマンドを実行します：

```bash
npx languine@latest translate
```

このコマンドを実行すると、Languineは以下の処理を行います：

- ソースJSONファイル（例：locales/en.json）を読み込む
- 新しいまたは変更された翻訳文字列を検出する
- 対象言語の翻訳を生成する
- 対象言語のJSONファイル（例：locales/sv.json、locales/de.json、locales/fr.json）を作成または更新する
