# CLI

Languineは、アプリケーションの翻訳を支援するCLIを提供します。

## コマンドの概要

### translate

`translate`コマンドは、アプリケーションを翻訳するために使用されます。

```bash title="Terminal"
# アプリケーションの翻訳を開始します
languine translate

# オプション:
#   --source-locale    ソースロケールを指定します（デフォルト: en）
#   --target-locale    翻訳先のターゲットロケールを指定します
#   --config          languine.config.jsファイルへのパス
```

### init

```bash title="Terminal"
# 新しいLanguineプロジェクトを初期化します
languine init

# これにより:
# - languine.config.jsファイルが作成されます
# - プロジェクト構造が設定されます
# - 必要なディレクトリが初期化されます
```

### status

```bash title="Terminal"
# 翻訳のステータスを確認します
languine status

# 表示内容:
# - 現在の翻訳進捗
# - 欠落している翻訳
# - 翻訳統計
```

### validate

```bash title="Terminal"
# 翻訳ファイルを検証します
languine validate

# チェック内容:
# - ファイル構造
# - 欠落しているキー
# - フォーマットの一貫性
```

### export

```bash title="Terminal"
# 翻訳をエクスポートします
languine export

# オプション:
#   --format          エクスポート形式（json、yamlなど）
#   --output         出力ディレクトリ
```

### import

```bash title="Terminal"
# 外部ファイルから翻訳をインポートします
languine import

# オプション:
#   --source         ソースファイルまたはディレクトリ
#   --format         インポート形式（json、yamlなど）
```

## グローバルオプション

すべてのコマンドは、以下のグローバルオプションをサポートしています:

```bash title="Terminal"
# すべてのコマンドで利用可能なグローバルオプション
languine [command] [options]

# 一般的なオプション:
#   --help, -h       ヘルプ情報を表示します
#   --version, -v    バージョン情報を表示します
#   --verbose        詳細ログを有効にします
#   --config         設定ファイルのパスを指定します
```
