# JSON

Languine prend en charge les fichiers JSON comme format source avec les exigences courantes pour la pluralisation et l'interpolation. Ce format est utilisé dans de nombreux frameworks comme React, Vue, Ruby on Rails, et plus encore.


---


## Configuration

Tout d'abord, assurez-vous d'avoir un fichier de configuration languine.json à la racine de votre projet. Voici un exemple :

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Traduction

Avec votre configuration en place, exécutez :

```bash
npx languine@latest translate
```

Lorsque vous exécutez cette commande, Languine va :

- Charger votre fichier JSON source (par exemple, locales/en.json)
- Détecter toute nouvelle chaîne de traduction ou toute chaîne modifiée
- Générer des traductions pour vos langues cibles
- Créer ou mettre à jour les fichiers JSON des langues cibles (par exemple, locales/sv.json, locales/de.json, locales/fr.json)
