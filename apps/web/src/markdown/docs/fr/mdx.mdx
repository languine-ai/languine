# Markdown

Un fichier markdown traduit est traité comme un Document, et est traduit dans son ensemble pour préserver le contexte du document.

## Ignorer le contenu

Vous pouvez ignorer le contenu entre les balises `<!-- languine-ignore-start -->` et `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Ce contenu ne sera pas traduit.
<!-- languine-ignore-end -->
```
