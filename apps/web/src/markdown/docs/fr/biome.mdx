# Biome

Languine prend en charge le formatage avec [Biome](https://biomejs.dev/) via des hooks. Vous pouvez configurer Biome pour formater automatiquement vos fichiers de traduction après leur génération.

Voici un exemple de configuration :

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Formater les fichiers générés avec Biome
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
