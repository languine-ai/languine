# JSON

לנגווין תומך בקבצי JSON כפורמט מקור עם הדרישות הנפוצות לריבוי והכנסה. פורמט זה משמש בהרבה מסגרות כמו React, Vue, Ruby on Rails ועוד.


---


## הגדרת מערכת

ראשית, ודא שיש לך קובץ הגדרות languine.json או languine.ts בשורש הפרויקט שלך. הנה דוגמה:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## תרגום

עם ההגדרות שלך, הרץ:

```bash
npx languine@latest translate
```

כאשר אתה מריץ את הפקודה הזו, לנגווין:

- טוען את קובץ ה-JSON המקורי שלך (למשל, locales/en.json)
- מזהה כל מיתר תרגום חדש או שונה
- מייצר תרגומים לשפות היעד שלך
- יוצר או מעדכן את קבצי ה-JSON של שפת היעד (למשל, locales/sv.json, locales/de.json, locales/fr.json)
