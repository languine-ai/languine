# התחלה מהירה

התחל בהתקנת Languine.

לאחר מכן, הרץ את הפקודה הבאה כדי לאתחל את Languine.

```bash title="טרמינל"
npx languine@latest init
```

עקוב אחרי הצעדים כדי להגדיר את הפרויקט שלך.

## תרגום ראשון

עכשיו אתה מוכן לתרגם את הקוד שלך.

התחל על ידי הוספת מפתח לקובץ המקור שלך, לדוגמה אם אתה בוחר להשתמש ב-`en.json` כקובץ המקור שלך, הוסף לו מפתח.

```json title="src/locales/en.json"
{
  "welcome": "ברוך הבא לאפליקציה שלנו!",
  "user": {
    "greeting": "שלום, {{name}}!",
    "profile": {
      "title": "הפרופיל שלך",
      "edit": "ערוך פרופיל"
    }
  },
  "notifications": {
    "messages": "יש לך {{count}} הודעה/הודעות חדשות.",
    "empty": "אין התראות חדשות."
  }
}

```


## תרגום

הרץ את הפקודה הבאה כדי לתרגם את התוכן שלך:

```bash title="טרמינל"
npx languine@latest translate
```

## זה יגרום ל:
1. ליצור קבצי תרגום לכל שפת יעד
2. לייצר תרגומים באיכות גבוהה באמצעות מנוע Languine
3. ליצור קובץ `languine.lock` כדי לעקוב אחרי מצבי התרגום

קובץ הנעילה עוזר ל-Languine לזהות אילו תרגומים צריכים עדכונים כאשר התוכן המקורי שלך משתנה. ודא שאתה מחויב לקובץ זה במאגר שלך.

## צעדים הבאים

עכשיו כשיש לך את Languine מוגדר, אתה יכול:
- להוסיף שפות נוספות בקובץ `languine.json`
- לשלב את Languine בצינור ה-CI/CD שלך
- להשתמש ב-API שלנו לתרגומים תוכנתיים

בדוק את המדריכים האחרים שלנו כדי ללמוד עוד על תכונות מתקדמות של Languine.
