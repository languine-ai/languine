# אימות

כדי להשתמש בלנגוין, עליך לאמת את עצמך עם המנוע שלנו, זה לוקח רק כמה שניות להתחיל.

---

## התחברות

אתה יכול להתחבר ישירות ב-[לוח הבקרה של לנגוין](https://languine.ai/login) או להריץ את הפקודה הבאה:

```bash title="טרמינל"
npx languine@latest auth login
```


אם כבר יש לך חשבון, תתחבר אוטומטית ומושב יישמר במחשב שלך.

---

## התנתקות

אתה יכול להתנתק על ידי הרצת הפקודה הבאה:

```bash title="טרמינל"
npx languine@latest auth logout
```

---

## מושב משתמש

אתה יכול לבדוק אם אתה מחובר על ידי הרצת הפקודה הבאה:

```bash title="טרמינל"
npx languine@latest auth whoami
```

---


# משתני סביבה

אתה יכול להגדיר את משתני הסביבה הבאים כדי לקבוע את הגדרות לנגוין:

- `LANGUINE_API_KEY`: מפתח ה-API לשימוש באימות.
- `LANGUINE_PROJECT_ID`: מזהה הפרויקט לשימוש בפרויקט.

אתה יכול למצוא אותם ב-[לוח הבקרה של לנגוין](https://languine.ai/login).
