# CI/CD

לנגווין מספקת אינטגרציה CI/CD כדי לעזור לך לתרגם את היישום שלך.

---

## איך זה עובד

תהליך העבודה של CI/CD בלנגווין עוקב אחרי תהליך מובנה כדי לאוטומט את עדכוני התרגום ביישום שלך:

### 1. התחלת פעולה
- התהליך מתחיל כאשר הפעולה מופעלת
- מגדיר את הגדרות ה-Git והאישורים הנדרשים

### 2. עיבוד מנוע לנגווין
- מנוע לנגווין מעבד את התוכן של היישום שלך
- מטפל במשימות התרגום באמצעות ההגדרות המוגדרות

### 3. זיהוי שינויים
המערכת בודקת אם יש שינויים בתרגום ועוקבת אחרי אחת משתי דרכים:

**כאשר נמצאו שינויים:**
- השינויים נשמרים במאגר
- השינויים נדחפים לענף המוגדר
- בקשת משיכה (PR) נוצרת

**כאשר לא נמצאו שינויים:**
- התהליך מסתיים מבלי ליצור PR

### 4. יצירת בקשת משיכה
כאשר זוהו שינויים, נוצרת בקשת משיכה עם:

**תוכן PR:**
- כותרת ברורה המצביעה על עדכוני תרגום
- תיאור מפורט של השינויים
- אישורי בדיקות איכות
- הוראות לסקירה
- קישורים לתיעוד ולתמיכה

**תיאור PR כולל:**
- סיכום של עדכוני התרגום
- רשימה של מה השתנה (תרגומים מעודכנים, עקביות במונחים)
- צעדים הבאים לסוקרים
- קישורים למשאבים של תיעוד ותמיכה
