# Autentificare

Pentru a utiliza Languine, trebuie să vă autentificați cu motorul nostru, durează doar câteva secunde pentru a începe.

---

## Conectare

Puteți să vă conectați direct pe [Languine Dashboard](https://languine.ai/login) sau să rulați următoarea comandă:

```bash title="Terminal"
npx languine@latest auth login
```


Dacă aveți deja un cont, veți fi autentificat automat și o sesiune va fi salvată pe mașina dumneavoastră.

---

## Deconectare

Puteți să vă deconectați rulând următoarea comandă:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Sesiune utilizator

Puteți verifica dacă sunteți autentificat rulând următoarea comandă:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Variabile de mediu

Puteți seta următoarele variabile de mediu pentru a configura Languine:

- `LANGUINE_API_KEY`: Cheia API de utilizat pentru autentificare.
- `LANGUINE_PROJECT_ID`: ID-ul proiectului de utilizat pentru proiect.

Le găsiți în [Languine Dashboard](https://languine.ai/login).
