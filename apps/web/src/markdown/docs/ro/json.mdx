# JSON

Languine suportă fișiere JSON ca format sursă cu cerințele comune pentru pluralizare și interpolare. Acest format este utilizat în multe framework-uri precum React, Vue, Ruby on Rails și altele.


---


## Configurare

În primul rând, asigură-te că ai un fișier de configurare languine.json sau languine.ts în rădăcina proiectului tău. Iată un exemplu:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Traducere

Cu configurația setată, rulează:

```bash
npx languine@latest translate
```

Când rulezi acest comandament, Languine va:

- Încărca fișierul tău JSON sursă (de exemplu, locales/en.json)
- Detecta orice șiruri de traducere noi sau modificate
- Genera traduceri pentru limbile tale țintă
- Crea sau actualiza fișierele JSON pentru limbile țintă (de exemplu, locales/sv.json, locales/de.json, locales/fr.json)
