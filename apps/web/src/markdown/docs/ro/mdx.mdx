# Markdown

Un fișier markdown tradus este tratat ca un Document și este tradus ca un întreg.

## Ignorarea conținutului

Puteți ignora conținutul dintre etichetele `<!-- languine-ignore-start -->` și `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Acest conținut nu va fi tradus.
<!-- languine-ignore-end -->
```
