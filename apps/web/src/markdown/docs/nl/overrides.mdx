# Overschrijvingen

Languine stelt je in staat om vertalingen voor specifieke sleutels te overschrijven. Dit is nuttig wanneer je wilt zorgen dat een specifieke vertaling wordt gebruikt voor een bepaalde sleutel, ongeacht de standaardvertaling.

## Hoe het werkt

In het dashboard kun je overschrijvingen maken voor specifieke sleutels. Je kunt ook overschrijvingen maken voor specifieke talen.
