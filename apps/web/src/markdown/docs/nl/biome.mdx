# Biome

Languine ondersteunt opmaak met [Biome](https://biomejs.dev/) via hooks. Je kunt Biome configureren om je vertaalbestanden automatisch op te maken nadat ze zijn gegenereerd.

Hier is een voorbeeldconfiguratie:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Genereer bestanden op met Biome
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
