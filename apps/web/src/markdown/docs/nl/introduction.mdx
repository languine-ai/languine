# Inleiding

Languine is een open-source lokalisatieplatform dat ontwikkelaars helpt hun applicaties te vertalen met AI-gestuurde vertalingen. Het biedt een naadloze workflow voor het beheren van vertalingen in meerdere talen, terwijl het hoge kwaliteit behoudt door middel van machine learning.

## Belangrijkste Kenmerken

- **AI-gestuurde Vertalingen**: Maak gebruik van geavanceerde machine learning-modellen om hoogwaardige vertalingen te genereren in meer dan 100 talen
- **Git-integratie**: Synchroniseer uw vertalingen direct met uw Git-repositories en automatiseer updates via CI/CD-pijplijnen
- **Ontwikkelaar-gericht**: Gebouwd met ontwikkelaars in gedachten, met een krachtige CLI, API en GitHub Action-integratie
- **Aangepaste Parsers**: Ondersteuning voor verschillende bestandsformaten en de mogelijkheid om aangepaste parsers te maken voor uw specifieke behoeften
- **Versiebeheer**: Volg wijzigingen, beheer takken en behoud de vertaalgeschiedenis
- **Open Source**: Volledig open source en beschikbaar op GitHub, wat transparantie en bijdragen van de gemeenschap mogelijk maakt

## Hoe Het Werkt

1. **Instellen**: Installeer de Languine CLI en configureer uw project met een eenvoudig initialisatieproces
2. **Verbinden**: Koppel uw repository en stel uw voorkeursworkflow in (CLI, API of GitHub Action)
3. **Vertalen**: Push uw brontekst en laat Languine de vertalingen naar uw doeltalen afhandelen
4. **Synchroniseren**: Synchroniseer automatisch de vertaalde inhoud terug naar uw repository en implementeer met vertrouwen
