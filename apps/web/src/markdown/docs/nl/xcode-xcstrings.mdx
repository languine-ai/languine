# Xcode Strings (XCStrings)

Languine ondersteunt het XCStrings-formaat van Xcode, het moderne lokalisatieformaat voor iOS- en macOS-toepassingen. Dit formaat beheert stringcatalogi met ondersteuning voor variaties, pluralisatie en opmerkingen van ontwikkelaars.

---

## Instellen

Zorg er eerst voor dat je een languine.json-configuratiebestand in de hoofdmap van je project hebt. Hier is een voorbeeld:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "xcstrings": {
      "include": ["MyApp/Localizable.xcstrings"]
    }
  }
}
```

## Vertalen

Met je configuratie ingesteld, voer je uit:

```bash
npx languine@latest translate
```

Wanneer je dit commando uitvoert, zal Languine:

- Je XCStrings-catalogus laden
- Nieuwe of gewijzigde strings detecteren
- Vertalingen genereren voor je doeltalen
- De stringcatalogus bijwerken met nieuwe vertalingen
- Stringvariaties en pluralisatieregels behouden
- Opmerkingen van ontwikkelaars en context behouden
- Variabele substituties verwerken
- Adaptieve strings ondersteunen
- Formatteringsspecificaties behouden
- De structuur van de stringcatalogus valideren
