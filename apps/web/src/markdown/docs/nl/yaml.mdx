# YAML

Languine ondersteunt YAML-bestanden als een bronformaat, dat veel wordt gebruikt in Ruby on Rails, Laravel en andere frameworks. Dit formaat ondersteunt geneste structuren, arrays en complexe gegevenstypen.

---

## Instellen

Zorg er eerst voor dat je een languine.json-configuratiebestand in de hoofdmap van je project hebt. Hier is een voorbeeld:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "yaml": {
      "include": ["config/locales/[locale].yml"]
    }
  }
}
```

## Vertalen

Met je configuratie ingesteld, voer je uit:

```bash
npx languine@latest translate
```

Wanneer je dit commando uitvoert, zal Languine:

- Je bron-YAML-bestand laden (bijv. config/locales/en.yml)
- Nieuwe of gewijzigde vertaalstrings detecteren
- Vertalingen genereren voor je doeltalen
- De YAML-bestanden voor de doeltaal aanmaken of bijwerken (bijv. config/locales/sv.yml)
- Geneste structuren en opmaak behouden
