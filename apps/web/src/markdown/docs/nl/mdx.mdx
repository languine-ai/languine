# Markdown

Een vertaald markdown-bestand wordt behandeld als een Document en wordt als geheel vertaald om de context van het document te behouden.

## Inhoud negeren

Je kunt inhoud negeren tussen de `<!-- languine-ignore-start -->` en `<!-- languine-ignore-end -->` tags.

```mdx
<!-- languine-ignore-start -->
Deze inhoud zal niet worden vertaald.
<!-- languine-ignore-end -->
```
