# CLI

Languine biedt een CLI om je applicatie te vertalen.

## Overzicht van commando's

### vertalen

Het `vertalen` commando wordt gebruikt om je applicatie te vertalen.

```bash title="Terminal"
# Start de vertaling van je applicatie
languine vertalen

# Opties:
#   --source-locale    Geef de brontalen op (standaard: en)
#   --target-locale    Geef de doeltaal(en) op om naar te vertalen
#   --config          Pad naar je languine.config.js bestand
```

### init

```bash title="Terminal"
# Initialiseer een nieuw Languine-project
languine init

# Dit zal:
# - Een languine.config.js bestand aanmaken
# - Je projectstructuur opzetten
# - Noodzakelijke mappen initialiseren
```

### status

```bash title="Terminal"
# Controleer de status van je vertalingen
languine status

# Toont:
# - Huidige vertaalvoortgang
# - Ontbrekende vertalingen
# - Vertaalstatistieken
```

### valideren

```bash title="Terminal"
# Valideer je vertaalbestanden
languine valideren

# Controleert:
# - Bestandsstructuur
# - Ontbrekende sleutels
# - Consistentie van het formaat
```

### exporteren

```bash title="Terminal"
# Exporteer je vertalingen
languine exporteren

# Opties:
#   --format          Exportformaat (json, yaml, enz.)
#   --output         Uitvoer directory
```

### importeren

```bash title="Terminal"
# Importeer vertalingen vanuit externe bestanden
languine importeren

# Opties:
#   --source         Bronbestand of directory
#   --format         Importformaat (json, yaml, enz.)
```

## Globale opties

Alle commando's ondersteunen de volgende globale opties:

```bash title="Terminal"
# Globale opties beschikbaar voor alle commando's
languine [commando] [opties]

# Veelvoorkomende opties:
#   --help, -h       Toon hulpinformatie
#   --version, -v    Toon versie-informatie
#   --verbose        Schakel gedetailleerde logging in
#   --config         Geef het pad naar het configuratiebestand op
```
