# XLIFF

Languine ondersteunt XLIFF (XML Localization Interchange File Format), een op XML gebaseerd formaat dat is ontworpen voor het uitwisselen van lokalisatiegegevens. Dit formaat wordt veel gebruikt in professionele vertaalworkflows en ondersteunt zowel XLIFF 1.2 als 2.0 versies.

---

## Instellen

Zorg er eerst voor dat je een languine.config.json of languine.config.ts configuratiebestand in de hoofdmap van je project hebt. Hier is een voorbeeld:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "xliff": {
      "include": ["locales/[locale].xlf"]
    }
  }
}
```

## Vertalen

Met je configuratie ingesteld, voer je uit:

```bash
npx languine@latest translate
```

Wanneer je dit commando uitvoert, zal Languine:

- Je bron XLIFF-bestand laden (bijv. locales/en.xlf)
- Nieuwe of gewijzigde vertaalunits detecteren
- Vertalingen genereren voor je doeltalen
- De XLIFF-bestanden voor de doeltaal aanmaken of bijwerken (bijv. locales/sv.xlf)
- Aantekeningen, contextinformatie en statusattributen behouden
- Compatibiliteit met CAT (Computer-Aided Translation) tools handhaven
