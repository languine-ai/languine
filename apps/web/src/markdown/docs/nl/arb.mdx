# ARB (Application Resource Bundle)

Languine ondersteunt ARB-bestanden, het standaardformaat voor lokalisatie van Flutter-toepassingen. Dit formaat behandelt meervoudsvormen, tijdelijke aanduidingen en selectie-uitdrukkingen terwijl het compatibel blijft met het intl-pakket van Flutter.

---

## Instellen

Zorg er eerst voor dat je een languine.json configuratiebestand hebt in de hoofddirectory van je project. Hier is een voorbeeld:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "arb": {
      "include": ["lib/l10n/intl_[locale].arb"]
    }
  }
}
```

## Vertalen

Met je configuratie ingesteld, voer je het volgende uit:

```bash
npx languine@latest translate
```

Wanneer je dit commando uitvoert, zal Languine:

- Je bron-ARB-bestand laden (bijv. lib/l10n/intl_en.arb)
- Nieuwe of gewijzigde vertaalstrings detecteren
- Vertalingen genereren voor je doeltalen
- De ARB-bestanden voor de doeltalen aanmaken of bijwerken (bijv. lib/l10n/intl_sv.arb)
- Metadata, tijdelijke aanduidingen en meervoud/selectie-uitdrukkingen behouden
