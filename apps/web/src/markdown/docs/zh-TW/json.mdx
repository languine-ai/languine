# JSON

Languine 支援 JSON 檔案作為來源格式，具備常見的複數化和插值需求。此格式在許多框架中使用，如 React、Vue、Ruby on Rails 等等。


---


## 設定

首先，請確保您的專案根目錄中有一個 languine.json 或 languine.ts 配置檔。以下是一個範例：

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## 翻譯

配置完成後，執行：

```bash
npx languine@latest translate
```

當您執行此命令時，Languine 將會：

- 載入您的來源 JSON 檔案（例如：locales/en.json）
- 偵測任何新的或已修改的翻譯字串
- 生成目標語言的翻譯
- 創建或更新目標語言的 JSON 檔案（例如：locales/sv.json、locales/de.json、locales/fr.json）
