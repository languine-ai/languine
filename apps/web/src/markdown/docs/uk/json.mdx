# JSON

Languine підтримує файли JSON як вихідний формат з загальними вимогами до множинності та інтерполяції. Цей формат використовується в багатьох фреймворках, таких як React, Vue, Ruby on Rails та інших.


---


## Налаштування

Спочатку переконайтеся, що у вас є файл конфігурації languine.json або languine.ts у кореневій папці вашого проєкту. Ось приклад:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Переклад

З налаштованим конфігурацією запустіть:

```bash
npx languine@latest translate
```

Коли ви виконаєте цю команду, Languine:

- Завантажить ваш вихідний JSON файл (наприклад, locales/en.json)
- Виявить нові або змінені рядки перекладу
- Згенерує переклади для ваших цільових мов
- Створить або оновить файли JSON для цільової мови (наприклад, locales/sv.json, locales/de.json, locales/fr.json)
