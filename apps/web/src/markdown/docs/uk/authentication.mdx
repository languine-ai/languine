# Аутентифікація

Щоб використовувати Languine, вам потрібно автентифікуватися в нашому Engine, це займає лише кілька секунд, щоб почати.

---

## Увійти

Ви можете увійти безпосередньо на [Languine Dashboard](https://languine.ai/login) або виконати наступну команду:

```bash title="Термінал"
npx languine@latest auth login
```


Якщо у вас вже є обліковий запис, ви будете автоматично увійдені, і сесія буде збережена на вашому комп'ютері.

---

## Вийти

Ви можете вийти, виконавши наступну команду:

```bash title="Термінал"
npx languine@latest auth logout
```

---

## Сесія користувача

Ви можете перевірити, чи ви увійшли, виконавши наступну команду:

```bash title="Термінал"
npx languine@latest auth whoami
```

---


# Змінні середовища

Ви можете встановити наступні змінні середовища для налаштування Languine:

- `LANGUINE_API_KEY`: API-ключ для використання для автентифікації.
- `LANGUINE_PROJECT_ID`: ID проекту для використання в проекті.

Ви знайдете їх у [Languine Dashboard](https://languine.ai/login).
