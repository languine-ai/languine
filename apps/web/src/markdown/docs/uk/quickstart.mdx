# Швидкий старт

Почніть з установки Languine.

Потім виконайте наступну команду для ініціалізації Languine.

```bash title="Термінал"
npx languine@latest init
```

Слідуйте інструкціям, щоб налаштувати ваш проект.

## Переклад 1

Тепер ви готові перекладати свій код.

Почніть з додавання ключа до вашого вихідного файлу, наприклад, якщо ви вирішите використовувати `en.json` як ваш вихідний файл, додайте до нього ключ.

```json title="src/locales/en.json"
{
  "welcome": "Ласкаво просимо до нашого додатку!",
  "user": {
    "greeting": "Привіт, {{name}}!",
    "profile": {
      "title": "Ваш профіль",
      "edit": "Редагувати профіль"
    }
  },
  "notifications": {
    "messages": "У вас {{count}} нове(і) повідомлення(я).",
    "empty": "Немає нових сповіщень."
  }
}

```


## Переклад

Виконайте наступну команду, щоб перекласти ваш контент:

```bash title="Термінал"
npx languine@latest translate
```

## Це зробить:
1. Створить файли перекладу для кожної цільової мови
2. Згенерує високоякісні переклади за допомогою Languine Engine
3. Створить файл `languine.lock` для відстеження станів перекладу

Файл блокування допомагає Languine визначити, які переклади потребують оновлення, коли ваш вихідний контент змінюється. Переконайтеся, що ви комітите цей файл у ваш репозиторій.

## Наступні кроки

Тепер, коли ви налаштували Languine, ви можете:
- Додати більше мов у ваш файл `languine.json`
- Інтегрувати Languine у ваш CI/CD конвеєр
- Використовувати наше API для програмних перекладів

Перегляньте наші інші посібники, щоб дізнатися більше про розширені функції Languine.
