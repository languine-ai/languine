# Маркування

Перекладений файл маркування розглядається як документ і перекладається цілком, щоб зберегти контекст документа.

## Ігнорування вмісту

Ви можете ігнорувати вміст між тегами `<!-- languine-ignore-start -->` та `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Цей вміст не буде перекладено.
<!-- languine-ignore-end -->
```
