# Markdown

Un fitxer markdown traduït es tracta com un Document, i es tradueix en la seva totalitat.

## Ignorant contingut

Podeu ignorar el contingut entre les etiquetes `<!-- languine-ignore-start -->` i `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Aquest contingut no serà traduït.
<!-- languine-ignore-end -->
```
