# Inici ràpid

Comença instal·lant Languine.

A continuació, executa la següent comanda per inicialitzar Languine.

```bash title="Terminal"
npx languine@latest init
```

Segueix els passos per configurar el teu projecte.

## Primera traducció

Ara estàs llest per traduir el teu codi.

Comença afegint una clau al teu fitxer font, per exemple, si decideixes utilitzar `en.json` com a fitxer font, afegeix una clau a ell.

```json title="src/locales/en.json"
{
  "welcome": "Benvingut a la nostra aplicació!",
  "user": {
    "greeting": "Hola, {{name}}!",
    "profile": {
      "title": "El teu perfil",
      "edit": "Edita el perfil"
    }
  },
  "notifications": {
    "messages": "Tens {{count}} nou(s) missatge(s).",
    "empty": "No hi ha noves notificacions."
  }
}

```


## Tradueix

Executa la següent comanda per traduir el teu contingut:

```bash title="Terminal"
npx languine@latest translate
```

## Això farà:
1. Crear fitxers de traducció per a cada idioma de destinació
2. Generar traduccions d'alta qualitat amb el motor Languine
3. Crear un fitxer `languine.lock` per fer un seguiment dels estats de traducció

El fitxer de bloqueig ajuda Languine a identificar quines traduccions necessiten actualitzacions quan el teu contingut font canvia. Assegura't de fer un commit d'aquest fitxer al teu repositori.

## Properes passes

Ara que tens Languine configurat, pots:
- Afegir més idiomes al teu fitxer `languine.json`
- Integrar Languine al teu pipeline de CI/CD
- Utilitzar la nostra API per a traduccions programàtiques

Consulta les nostres altres guies per aprendre més sobre les funcions avançades de Languine.
