# Autenticació

Per utilitzar Languine, necessites autenticar-te amb el nostre Engine, només triga uns segons a començar.

---

## Iniciar sessió

Pots iniciar sessió directament al [Tauler de Languine](https://languine.ai/login) o executar la següent comanda:

```bash title="Terminal"
npx languine@latest auth login
```


Si ja tens un compte, iniciaràs sessió automàticament i es guardarà una sessió al teu ordinador.

---

## Tancar sessió

Pots tancar sessió executant la següent comanda:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Sessió d'usuari

Pots comprovar si estàs iniciat sessió executant la següent comanda:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Variables d'entorn

Pots establir les següents variables d'entorn per configurar Languine:

- `LANGUINE_API_KEY`: La clau API per utilitzar per a l'autenticació.
- `LANGUINE_PROJECT_ID`: L'ID del projecte per utilitzar per al projecte.

Les trobaràs al [Tauler de Languine](https://languine.ai/login).
