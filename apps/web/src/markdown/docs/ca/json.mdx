# JSON

Languine suporta fitxers JSON com a format font amb els requisits comuns per a la pluralització i la interpolació. Aquest format s'utilitza en molts marcs com React, Vue, Ruby on Rails, i més.


---


## Configuració

Primer, assegureu-vos que teniu un fitxer de configuració languine.json o languine.ts a la carpeta arrel del vostre projecte. Aquí teniu un exemple:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Traducció

Amb la vostra configuració establerta, executeu:

```bash
npx languine@latest translate
```

Quan executeu aquest comandament, Languine:

- Carregarà el vostre fitxer JSON font (per exemple, locales/en.json)
- Detectarà qualsevol cadena de traducció nova o modificada
- Generarà traduccions per a les vostres llengües objectiu
- Crearà o actualitzarà els fitxers JSON de les llengües objectiu (per exemple, locales/sv.json, locales/de.json, locales/fr.json)
