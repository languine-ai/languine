# אימות

כדי להשתמש ב-Languine, עליך לאמת את עצמך עם המנוע שלנו, זה לוקח רק כמה שניות להתחיל.

---

## התחברות

אתה יכול להתחבר ישירות ב-[Languine Dashboard](https://languine.ai/login) או להריץ את הפקודה הבאה:

```bash title="Terminal"
npx languine@latest auth login
```


אם כבר יש לך חשבון, תתחבר אוטומטית והסשן יישמר במחשב שלך.

---

## התנתקות

אתה יכול להתנתק על ידי הרצת הפקודה הבאה:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## סשן משתמש

אתה יכול לבדוק אם אתה מחובר על ידי הרצת הפקודה הבאה:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# משתני סביבה

אתה יכול להגדיר את משתני הסביבה הבאים כדי לקונפג את Languine:

- `LANGUINE_API_KEY`: מפתח ה-API לשימוש באימות.
- `LANGUINE_PROJECT_ID`: מזהה הפרויקט לשימוש בפרויקט.

אתה יכול למצוא אותם ב-[Languine Dashboard](https://languine.ai/login).
