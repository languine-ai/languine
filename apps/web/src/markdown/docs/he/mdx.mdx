# Markdown

קובץ markdown מתורגם נחשב למסמך, ומתרגם אותו כולו כדי לשמור על ההקשר של המסמך.

## התעלמות מתוכן

אתה יכול להתעלם מתוכן בין תגיות `<!-- languine-ignore-start -->` ל- `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
תוכן זה לא יתורגם.
<!-- languine-ignore-end -->
```
