# Markdown

קובץ markdown מתורגם נחשב למסמך, והוא מתורגם במלואו כדי לשמור על הקשר של המסמך.

## התעלמות מתוכן

אתה יכול להתעלם מתוכן בין תגיות `<!-- languine-ignore-start -->` ו- `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
תוכן זה לא יתורגם.
<!-- languine-ignore-end -->
```
