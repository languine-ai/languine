# JSON

Languine поддерживает файлы JSON в качестве исходного формата с общими требованиями к плюрализации и интерполяции. Этот формат используется во многих фреймворках, таких как React, Vue, Ruby on Rails и других.


---


## Настройка

Сначала убедитесь, что у вас есть файл конфигурации languine.json или languine.ts в корне вашего проекта. Вот пример:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Перевод

С установленной конфигурацией выполните:

```bash
npx languine@latest translate
```

Когда вы выполните эту команду, Languine:

- Загрузит ваш исходный файл JSON (например, locales/en.json)
- Обнаружит любые новые или измененные строки перевода
- Сгенерирует переводы для ваших целевых языков
- Создаст или обновит файлы JSON для целевого языка (например, locales/sv.json, locales/de.json, locales/fr.json)
