# CLI

Languine предоставляет интерфейс командной строки (CLI), чтобы помочь вам перевести ваше приложение.

## Обзор команд

### translate

Команда `translate` используется для перевода вашего приложения.

```bash title="Terminal"
# Начать перевод вашего приложения
languine translate

# Опции:
#   --source-locale    Укажите исходный язык (по умолчанию: en)
#   --target-locale    Укажите целевой язык(и) для перевода
#   --config          Путь к вашему файлу languine.config.js
```

### init

```bash title="Terminal"
# Инициализировать новый проект Languine
languine init

# Это приведет к:
# - Созданию файла languine.config.js
# - Настройке структуры вашего проекта
# - Инициализации необходимых директорий
```

### status

```bash title="Terminal"
# Проверить статус ваших переводов
languine status

# Показывает:
# - Текущий прогресс перевода
# - Недостающие переводы
# - Статистику перевода
```

### validate

```bash title="Terminal"
# Проверить ваши файлы перевода
languine validate

# Проверяет:
# - Структуру файлов
# - Недостающие ключи
# - Согласованность формата
```

### export

```bash title="Terminal"
# Экспортировать ваши переводы
languine export

# Опции:
#   --format          Формат экспорта (json, yaml и т.д.)
#   --output         Выходной каталог
```

### import

```bash title="Terminal"
# Импортировать переводы из внешних файлов
languine import

# Опции:
#   --source         Исходный файл или каталог
#   --format         Формат импорта (json, yaml и т.д.)
```

## Глобальные опции

Все команды поддерживают следующие глобальные опции:

```bash title="Terminal"
# Глобальные опции, доступные для всех команд
languine [command] [options]

# Общие опции:
#   --help, -h       Показать справочную информацию
#   --version, -v    Показать информацию о версии
#   --verbose        Включить подробный лог
#   --config         Указать путь к файлу конфигурации
```
