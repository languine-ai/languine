# Быстрый старт

Начните с установки Languine.

Затем выполните следующую команду для инициализации Languine.

```bash title="Терминал"
npx languine@latest init
```

Следуйте инструкциям, чтобы настроить ваш проект.

## Перевод

Теперь вы готовы перевести ваш код.

Начните с добавления ключа в ваш исходный файл, например, если вы выбрали использовать `en.json` в качестве исходного файла, добавьте в него ключ.

```json title="src/locales/en.json"
{
  "welcome": "Добро пожаловать в наше приложение!",
  "user": {
    "greeting": "Привет, {{name}}!",
    "profile": {
      "title": "Ваш профиль",
      "edit": "Редактировать профиль"
    }
  },
  "notifications": {
    "messages": "У вас {{count}} новое(ых) сообщение(й).",
    "empty": "Нет новых уведомлений."
  }
}

```


## Перевести

Выполните следующую команду для перевода вашего контента:

```bash title="Терминал"
npx languine@latest translate
```

## Это сделает:
1. Создаст файлы перевода для каждого целевого языка
2. Сгенерирует высококачественные переводы с помощью Languine Engine
3. Создаст файл `languine.lock` для отслеживания состояний перевода

Файл блокировки помогает Languine определить, какие переводы нуждаются в обновлении, когда ваш исходный контент изменяется. Убедитесь, что вы добавили этот файл в ваш репозиторий.

## Следующие шаги

Теперь, когда вы настроили Languine, вы можете:
- Добавить больше языков в ваш файл `languine.json`
- Интегрировать Languine в ваш CI/CD процесс
- Использовать наш API для программных переводов

Посмотрите наши другие руководства, чтобы узнать больше о продвинутых функциях Languine.
