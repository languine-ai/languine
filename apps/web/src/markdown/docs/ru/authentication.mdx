# Аутентификация

Чтобы использовать Languine, вам необходимо пройти аутентификацию в нашем движке, это займет всего несколько секунд, чтобы начать.

---

## Войти

Вы можете войти напрямую на [Панели управления Languine](https://languine.ai/login) или выполнить следующую команду:

```bash title="Терминал"
npx languine@latest auth login
```


Если у вас уже есть учетная запись, вы будете автоматически авторизованы, и сессия будет сохранена на вашем устройстве.

---

## Выйти

Вы можете выйти, выполнив следующую команду:

```bash title="Терминал"
npx languine@latest auth logout
```

---

## Сессия пользователя

Вы можете проверить, авторизованы ли вы, выполнив следующую команду:

```bash title="Терминал"
npx languine@latest auth whoami
```

---


# Переменные окружения

Вы можете установить следующие переменные окружения для настройки Languine:

- `LANGUINE_API_KEY`: Ключ API для аутентификации.
- `LANGUINE_PROJECT_ID`: Идентификатор проекта для использования в проекте.

Вы можете найти их на [Панели управления Languine](https://languine.ai/login).
