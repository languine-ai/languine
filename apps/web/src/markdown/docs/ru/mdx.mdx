# Маркдаун

Переведенный файл markdown рассматривается как документ и переводится целиком.

## Игнорирование содержимого

Вы можете игнорировать содержимое между тегами `<!-- languine-ignore-start -->` и `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Этот контент не будет переведен.
<!-- languine-ignore-end -->
```
