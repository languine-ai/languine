# Маркдаун

Переведённый файл в формате Markdown рассматривается как документ и переводится целиком, чтобы сохранить контекст документа.

## Игнорирование контента

Вы можете игнорировать контент между тегами `<!-- languine-ignore-start -->` и `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Этот контент не будет переведён.
<!-- languine-ignore-end -->
```
