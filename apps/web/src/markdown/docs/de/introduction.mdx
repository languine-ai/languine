# Einführung

Languine ist eine Open-Source-Lokalisierungsplattform, die Entwicklern hilft, ihre Anwendungen mit KI-gestützten Übersetzungen zu übersetzen. Sie bietet einen nahtlosen Workflow zur Verwaltung von Übersetzungen in mehreren Sprachen und gewährleistet dabei hohe Qualität durch maschinelles Lernen.

## Hauptmerkmale

- **KI-gestützte Übersetzungen**: Nutzen Sie fortschrittliche maschinelle Lernmodelle, um qualitativ hochwertige Übersetzungen in über 100 Sprachen zu erstellen
- **Git-Integration**: Synchronisieren Sie Ihre Übersetzungen direkt mit Ihren Git-Repositories und automatisieren Sie Updates über CI/CD-Pipelines
- **Entwicklerorientiert**: Entwickelt mit Blick auf Entwickler, mit einer leistungsstarken CLI, API und GitHub Action-Integration
- **Benutzerdefinierte Parser**: Unterstützung für verschiedene Dateiformate und die Möglichkeit, benutzerdefinierte Parser für Ihre spezifischen Bedürfnisse zu erstellen
- **Versionskontrolle**: Änderungen verfolgen, Branches verwalten und die Übersetzungshistorie pflegen
- **Open Source**: Vollständig Open Source und auf GitHub verfügbar, was Transparenz und Community-Beiträge ermöglicht

## So funktioniert es

1. **Einrichtung**: Installieren Sie die Languine CLI und konfigurieren Sie Ihr Projekt mit einem einfachen Initialisierungsprozess
2. **Verbinden**: Verknüpfen Sie Ihr Repository und richten Sie Ihren bevorzugten Workflow (CLI, API oder GitHub Action) ein
3. **Übersetzen**: Übertragen Sie Ihren Quellinhalt und lassen Sie Languine die Übersetzungen in Ihre Zielsprache übernehmen
4. **Synchronisieren**: Synchronisieren Sie automatisch übersetzte Inhalte zurück in Ihr Repository und stellen Sie mit Vertrauen bereit
