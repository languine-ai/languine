# JSON

Languine unterstützt JSON-Dateien als Quellformat mit den gängigen Anforderungen für Pluralisierung und Interpolation. Dieses Format wird in vielen Frameworks wie React, Vue, Ruby on Rails und mehr verwendet.


---


## Einrichtung

Zuerst stellen Sie sicher, dass Sie eine languine.json-Konfigurationsdatei im Stammverzeichnis Ihres Projekts haben. Hier ist ein Beispiel:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Übersetzung

Mit Ihrer Konfiguration führen Sie aus:

```bash
npx languine@latest translate
```

Wenn Sie diesen Befehl ausführen, wird Languine:

- Ihre Quell-JSON-Datei laden (z. B. locales/en.json)
- Neue oder geänderte Übersetzungsstrings erkennen
- Übersetzungen für Ihre Zielsprache generieren
- Die JSON-Dateien der Zielsprache erstellen oder aktualisieren (z. B. locales/sv.json, locales/de.json, locales/fr.json)
