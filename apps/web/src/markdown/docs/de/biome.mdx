# Biome

Languine unterstützt die Formatierung mit [Biome](https://biomejs.dev/) über Hooks. Sie können Biome so konfigurieren, dass Ihre Übersetzungsdateien automatisch formatiert werden, nachdem sie generiert wurden.

Hier ist ein Beispiel für eine Konfiguration:

```typescript title="languine.config.ts"
import { defineConfig } from "languine";

export default defineConfig({
  locale: {
    source: "en",
    targets: ["es", "fr"],
  },
  files: {
    json: {
      include: ["locales/[locale].json"],
    },
  },
  hooks: {
    // Generierte Dateien mit Biome formatieren
    afterTranslate: async (files) => {
      const { check } = await import("@biomejs/biome");
      
      for (const file of files) {
        await check({
          files: [file],
          apply: true,
        });
      }
    },
  },
});
```
