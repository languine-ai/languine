# Markdown

Eine übersetzte Markdown-Datei wird als Dokument behandelt und als Ganzes übersetzt, um den Kontext des Dokuments zu bewahren.

## Ignorieren von Inhalten

Sie können Inhalte zwischen den `<!-- languine-ignore-start -->` und `<!-- languine-ignore-end -->` Tags ignorieren.

```mdx
<!-- languine-ignore-start -->
Dieser Inhalt wird nicht übersetzt.
<!-- languine-ignore-end -->
```
