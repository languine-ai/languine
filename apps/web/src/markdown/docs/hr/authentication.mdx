# Autentifikacija

Da biste koristili Languine, trebate se autentificirati s našim Engine-om, potrebno je samo nekoliko sekundi da započnete.

---

## Prijava

Možete se prijaviti izravno na [Languine Dashboard](https://languine.ai/login) ili pokrenuti sljedeću naredbu:

```bash title="Terminal"
npx languine@latest auth login
```


Ako već imate račun, automatski ćete se prijaviti i sesija će biti pohranjena na vašem računalu.

---

## Odjava

Možete se odjaviti pokretanjem sljedeće naredbe:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## Korisnička sesija

Možete provjeriti jeste li prijavljeni pokretanjem sljedeće naredbe:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# Varijable okruženja

Možete postaviti sljedeće varijable okruženja za konfiguraciju Languine-a:

- `LANGUINE_API_KEY`: API ključ koji se koristi za autentifikaciju.
- `LANGUINE_PROJECT_ID`: ID projekta koji se koristi za projekt.

Možete ih pronaći na [Languine Dashboard](https://languine.ai/login).
