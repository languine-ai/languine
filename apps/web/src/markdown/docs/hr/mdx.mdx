# Markdown

Prevedena markdown datoteka se tretira kao Dokument i prevodi se u cjelini.

## Ignoriranje sadržaja

Možete ignorirati sadržaj između `<!-- languine-ignore-start -->` i `<!-- languine-ignore-end -->` oznaka.

```mdx
<!-- languine-ignore-start -->
Ovaj sadržaj neće biti preveden.
<!-- languine-ignore-end -->
```
