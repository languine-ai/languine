# CLI

Languine pruža CLI za pomoć u prevođenju vaše aplikacije.

## Pregled naredbi

### translate

Naredba `translate` koristi se za prevođenje vaše aplikacije.

```bash title="Terminal"
# Pokrenite prevođenje vaše aplikacije
languine translate

# Opcije:
#   --source-locale    Odredite izvorni jezik (zadano: en)
#   --target-locale    Odredite ciljni jezik(e) za prevođenje
#   --config          Putanja do vašeg languine.config.js datoteke
```

### init

```bash title="Terminal"
# Inicijalizirajte novi Languine projekt
languine init

# Ovo će:
# - Stvoriti languine.config.js datoteku
# - Postaviti strukturu vašeg projekta
# - Inicijalizirati potrebne direktorije
```

### status

```bash title="Terminal"
# Provjerite status vaših prijevoda
languine status

# Prikazuje:
# - Trenutni napredak prevođenja
# - Nedostajući prijevodi
# - Statistiku prevođenja
```

### validate

```bash title="Terminal"
# Potvrdite svoje datoteke s prijevodima
languine validate

# Provjerava:
# - Strukturu datoteke
# - Nedostajuće ključeve
# - Dosljednost formata
```

### export

```bash title="Terminal"
# Izvezite svoje prijevode
languine export

# Opcije:
#   --format          Format izvoza (json, yaml, itd.)
#   --output         Izlazni direktorij
```

### import

```bash title="Terminal"
# Uvezite prijevode iz vanjskih datoteka
languine import

# Opcije:
#   --source         Izvorna datoteka ili direktorij
#   --format         Format uvoza (json, yaml, itd.)
```

## Globalne opcije

Sve naredbe podržavaju sljedeće globalne opcije:

```bash title="Terminal"
# Globalne opcije dostupne za sve naredbe
languine [naredba] [opcije]

# Uobičajene opcije:
#   --help, -h       Prikaži informacije o pomoći
#   --version, -v    Prikaži informacije o verziji
#   --verbose        Omogući detaljno bilježenje
#   --config         Odredite putanju do konfiguracijske datoteke
```
