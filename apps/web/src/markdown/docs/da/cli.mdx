# CLI

Languine tilbyder en CLI til at hjælpe dig med at oversætte din applikation.

---

## Kommandoversigt

### Oversæt

`translate`-kommandoen bruges til at oversætte din applikation.

```bash title="Terminal"
    npx languine@latest translate
```

---

### Init

`init`-kommandoen bruges til at initialisere et nyt Languine-projekt.

```bash title="Terminal"
    npx languine@latest init
```

---

### Auth

`auth`-kommandoen bruges til at godkende med Languine.

```bash title="Terminal"
    npx languine@latest auth
```

---

### Sync

`sync`-kommandoen bruges til at synkronisere din applikation med Languine.

```bash title="Terminal"
    npx languine@latest sync
```

---

### Locale Tilføj

`locale`-kommandoen bruges til at administrere din applikations lokaliteter.

```bash title="Terminal"
    npx languine@latest locale add pt,fr
```

---

### Locale Fjern

`locale remove`-kommandoen bruges til at fjerne en lokalitet fra din applikation.

```bash title="Terminal"
    npx languine@latest locale remove pt,fr
```
