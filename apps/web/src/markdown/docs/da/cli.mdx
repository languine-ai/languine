# CLI

Languine tilbyder en CLI til at hjælpe dig med at oversætte din applikation.

## Kommandoversigt

### oversæt

`oversæt`-kommandoen bruges til at oversætte din applikation.

```bash title="Terminal"
# Start oversættelsen af din applikation
languine oversæt

# Muligheder:
#   --source-locale    Angiv kildesprog (standard: en)
#   --target-locale    Angiv mål-sprog(e) at oversætte til
#   --config          Sti til din languine.config.js fil
```

### init

```bash title="Terminal"
# Initialiser et nyt Languine-projekt
languine init

# Dette vil:
# - Oprette en languine.config.js fil
# - Opsætte din projektstruktur
# - Initialisere nødvendige mapper
```

### status

```bash title="Terminal"
# Tjek status for dine oversættelser
languine status

# Viser:
# - Nuværende oversættelsesfremskridt
# - Manglende oversættelser
# - Oversættelsesstatistik
```

### valider

```bash title="Terminal"
# Valider dine oversættelsesfiler
languine valider

# Tjekker:
# - Filstruktur
# - Manglende nøgler
# - Formatkonsistens
```

### eksport

```bash title="Terminal"
# Eksporter dine oversættelser
languine eksport

# Muligheder:
#   --format          Eksportformat (json, yaml, osv.)
#   --output         Output-mappe
```

### import

```bash title="Terminal"
# Importer oversættelser fra eksterne filer
languine import

# Muligheder:
#   --source         Kildefil eller mappe
#   --format         Importformat (json, yaml, osv.)
```

## Globale muligheder

Alle kommandoer understøtter følgende globale muligheder:

```bash title="Terminal"
# Globale muligheder tilgængelige for alle kommandoer
languine [kommando] [muligheder]

# Almindelige muligheder:
#   --help, -h       Vis hjælpeinformation
#   --version, -v    Vis versionsinformation
#   --verbose        Aktivér detaljeret logning
#   --config         Angiv sti til konfigurationsfil
```
