# ការផ្ទៀងផ្ទាត់

ដើម្បីប្រើប្រាស់ Languine អ្នកត្រូវតែផ្ទៀងផ្ទាត់ជាមួយម៉ាស៊ីនរបស់យើង វាអាចចំណាយពេលតិចតួចក្នុងការចាប់ផ្តើម។

---

## ចូលគណនី

អ្នកអាចចូលគណនីដោយផ្ទាល់នៅលើ [Languine Dashboard](https://languine.ai/login) ឬរត់ពាក្យបញ្ជាដូចខាងក្រោម៖

```bash title="Terminal"
npx languine@latest auth login
```


បើអ្នកមានគណនីរួចហើយ អ្នកនឹងត្រូវបានចូលគណនីដោយស្វ័យប្រវត្តិ ហើយសម័យវិនិយោគនឹងត្រូវបានរក្សាទុកនៅលើម៉ាស៊ីនរបស់អ្នក។

---

## ចាកចេញ

អ្នកអាចចាកចេញដោយរត់ពាក្យបញ្ជាដូចខាងក្រោម៖

```bash title="Terminal"
npx languine@latest auth logout
```

---

## សម័យអ្នកប្រើ

អ្នកអាចពិនិត្យមើលថាតើអ្នកបានចូលគណនីរួចហើយឬនៅដោយរត់ពាក្យបញ្ជាដូចខាងក្រោម៖

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# អថេរ​បរិស្ថាន

អ្នកអាចកំណត់អថេរ​បរិស្ថានខាងក្រោមដើម្បីកំណត់ Languine:

- `LANGUINE_API_KEY`: កូនសោ API ដែលត្រូវប្រើសម្រាប់ការផ្ទៀងផ្ទាត់។
- `LANGUINE_PROJECT_ID`: អត្តសញ្ញាណគម្រោងដែលត្រូវប្រើសម្រាប់គម្រោង។

អ្នកអាចរកឃើញពួកវានៅក្នុង [Languine Dashboard](https://languine.ai/login)។
