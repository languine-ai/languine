# CLI

Languine ផ្តល់នូវ CLI ដើម្បីជួយអ្នកបកប្រែកម្មវិធីរបស់អ្នក។

## ការមើលទូទៅអំពីពាក្យបញ្ជា

### translate

ពាក្យបញ្ជា `translate` ត្រូវបានប្រើសម្រាប់បកប្រែកម្មវិធីរបស់អ្នក។

```bash title="Terminal"
# ចាប់ផ្តើមការបកប្រែកម្មវិធីរបស់អ្នក
languine translate

# ជម្រើស៖
#   --source-locale    កំណត់ភាសាដើម (លំនាំដើម: en)
#   --target-locale    កំណត់ភាសាគោលដៅដែលត្រូវបកប្រែទៅ
#   --config          ផ្លូវទៅកាន់ឯកសារ languine.config.js របស់អ្នក
```

### init

```bash title="Terminal"
# ចាប់ផ្តើមគម្រោង Languine ថ្មី
languine init

# នេះនឹង៖
# - បង្កើតឯកសារ languine.config.js
# - កំណត់រចនាសម្ព័ន្ធគម្រោងរបស់អ្នក
# - ចាប់ផ្តើមថតដែលចាំបាច់
```

### status

```bash title="Terminal"
# ពិនិត្យស្ថានភាពនៃការបកប្រែរបស់អ្នក
languine status

# បង្ហាញ៖
# - ការរីកចម្រើននៃការបកប្រែបច្ចុប្បន្ន
# - ការបកប្រែដែលខ្វះ
# - ស្ថិតិការបកប្រែ
```

### validate

```bash title="Terminal"
# ផ្ទៀងផ្ទាត់ឯកសារបកប្រែរបស់អ្នក
languine validate

# ពិនិត្យ៖
# - រចនាសម្ព័ន្ធឯកសារ
# - កូនសោដែលខ្វះ
# - ភាពស្របតាមទម្រង់
```

### export

```bash title="Terminal"
# នាំចេញការបកប្រែរបស់អ្នក
languine export

# ជម្រើស៖
#   --format          ទម្រង់នាំចេញ (json, yaml, ល។)
#   --output         ថតចេញ
```

### import

```bash title="Terminal"
# នាំចូលការបកប្រែពីឯកសារបរទេស
languine import

# ជម្រើស៖
#   --source         ឯកសារឬថតដើម
#   --format         ទម្រង់នាំចូល (json, yaml, ល។)
```

## ជម្រើសសកល

ពាក្យបញ្ជាទាំងអស់គាំទ្រជម្រើសសកលដូចខាងក្រោម៖

```bash title="Terminal"
# ជម្រើសសកលដែលអាចប្រើបានសម្រាប់ពាក្យបញ្ជាទាំងអស់
languine [command] [options]

# ជម្រើសទូទៅ៖
#   --help, -h       បង្ហាញព័ត៌មានជំនួយ
#   --version, -v    បង្ហាញព័ត៌មានកំណែ
#   --verbose        បើកការចុះកំណត់លម្អិត
#   --config         កំណត់ផ្លូវឯកសារកំណត់
```
