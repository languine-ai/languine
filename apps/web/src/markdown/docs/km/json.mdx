# JSON

Languine គាំទ្រទ្រង់ទ្រាយ JSON ជាទ្រង់ទ្រាយប្រភពដែលមានតម្រូវការទូទៅសម្រាប់ការបន្ថែមច្រើន និងការបញ្ចូល។ ទ្រង់ទ្រាយនេះត្រូវបានប្រើនៅក្នុងហេដ្ឋារចនាសម្ព័ន្ធជាច្រើនដូចជា React, Vue, Ruby on Rails និងផ្សេងៗទៀត។


---


## ការកំណត់

ដំបូង សូមប្រាកដថាអ្នកមានឯកសារ languine.json ឬ languine.ts នៅក្នុងរង្វង់គម្រោងរបស់អ្នក។ នេះគឺជាឧទាហរណ៍៖

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## ការប្រែសម្រួល

ជាមួយនឹងការកំណត់រចនាសម្ព័ន្ធរបស់អ្នក បានកំណត់រួចហើយ សូមរត់៖

```bash
npx languine@latest translate
```

នៅពេលដែលអ្នករត់ពាក្យបញ្ជានេះ Languine នឹង៖

- បញ្ចូលឯកសារ JSON ប្រភពរបស់អ្នក (ឧ. locales/en.json)
- សម្គាល់អត្ថបទប្រែសម្រួលថ្មី ឬបានកែប្រែ
- បង្កើតការប្រែសម្រួលសម្រាប់ភាសាគោលដៅរបស់អ្នក
- បង្កើត ឬធ្វើអាប់ដេតឯកសារ JSON ភាសាគោលដៅ (ឧ. locales/sv.json, locales/de.json, locales/fr.json)
