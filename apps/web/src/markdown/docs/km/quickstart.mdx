# Quickstart

ចាប់ផ្តើមដោយការតំឡើង Languine។

បន្ទាប់មក ប្រើពាក្យបញ្ជាដូចខាងក្រោមដើម្បីចាប់ផ្តើម Languine។

```bash title="Terminal"
npx languine@latest init
```

អនុវត្តន៍ជំហានដើម្បីកំណត់គម្រោងរបស់អ្នក។

## ការប្រែសម្រួលដំបូង

ឥឡូវនេះអ្នកបានរៀបចំសម្រាប់ការប្រែសម្រួលកូដរបស់អ្នក។

ចាប់ផ្តើមដោយបន្ថែមកូនសោទៅឯកសារដើមរបស់អ្នក ដូចជា ប្រសិនបើអ្នកជ្រើសរើសប្រើ `en.json` ជាឯកសារដើមរបស់អ្នក សូមបន្ថែមកូនសោទៅវា។

```json title="src/locales/en.json"
{
  "welcome": "សូមស្វាគមន៍មកកាន់កម្មវិធីរបស់យើង!",
  "user": {
    "greeting": "សួស្តី, {{name}}!",
    "profile": {
      "title": "ប្រវត្តិរូបរបស់អ្នក",
      "edit": "កែប្រែប្រវត្តិរូប"
    }
  },
  "notifications": {
    "messages": "អ្នកមានសារថ្មី {{count}} សារ។",
    "empty": "មិនមានការជូនដំណឹងថ្មីទេ។"
  }
}

```


## ប្រែសម្រួល

ប្រើពាក្យបញ្ជាដូចខាងក្រោមដើម្បីប្រែសម្រួលមាតិការបស់អ្នក៖

```bash title="Terminal"
npx languine@latest translate
```

## នេះនឹង៖
1. បង្កើតឯកសារប្រែសម្រួលសម្រាប់ភាសាគោលដៅនីមួយៗ
2. បង្កើតការប្រែសម្រួលដែលមានគុណភាពខ្ពស់ដោយប្រើ Languine Engine
3. បង្កើតឯកសារ `languine.lock` ដើម្បីតាមដានស្ថានភាពការប្រែសម្រួល

ឯកសារ lock នេះជួយឱ្យ Languine ស្គាល់ថាការប្រែសម្រួលណាដែលត្រូវការការអាប់ដេតនៅពេលមាតិកាដើមរបស់អ្នកបានផ្លាស់ប្តូរ។ សូមប្រាកដថាបានបញ្ជូនឯកសារនេះទៅក្នុងរ៉េពូស៊ីទូរទៅរបស់អ្នក។

## ជំហានបន្ទាប់

ឥឡូវនេះដែលអ្នកបានកំណត់ Languine រួចរាល់ អ្នកអាច៖
- បន្ថែមភាសាផ្សេងទៀតនៅក្នុងឯកសារ `languine.json`
- បញ្ចូល Languine ទៅក្នុងបណ្តាញ CI/CD របស់អ្នក
- ប្រើ API របស់យើងសម្រាប់ការប្រែសម្រួលដោយកម្មវិធី

សូមពិនិត្យមើលមគ្គុទេសក៍ផ្សេងទៀតរបស់យើងដើម្បីរៀនបន្ថែមអំពីលក្ខណៈពិសេសកម្រិតខ្ពស់របស់ Languine។
