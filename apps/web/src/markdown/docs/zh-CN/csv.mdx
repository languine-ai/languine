# CSV

Languine 支持 CSV（逗号分隔值）文件，以便于简单的翻译管理。此格式非常适合基于电子表格的工作流程，并且可以轻松与外部翻译工具集成。

---

## 设置

首先，请确保在项目根目录中有一个 languine.config.json 或 languine.config.ts 配置文件。以下是一个示例：

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "csv": {
      "include": ["translations/[locale].csv"]
    }
  }
}
```

## 翻译

配置完成后，运行：

```bash
npx languine@latest translate
```

当您运行此命令时，Languine 将：

- 加载您的源 CSV 文件（例如，translations/en.csv）
- 检测任何新的或修改过的翻译字符串
- 为目标语言生成翻译
- 创建或更新目标语言的 CSV 文件（例如，translations/sv.csv）
- 保持列结构和格式
- 保留任何额外的元数据列
