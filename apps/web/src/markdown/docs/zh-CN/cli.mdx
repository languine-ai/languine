# CLI

Languine 提供了一个 CLI 来帮助您翻译您的应用程序。

## 命令概述

### translate

`translate` 命令用于翻译您的应用程序。

```bash title="Terminal"
# 开始翻译您的应用程序
languine translate

# 选项：
#   --source-locale    指定源语言环境（默认：en）
#   --target-locale    指定要翻译的目标语言环境
#   --config          您的 languine.config.js 文件的路径
```

### init

```bash title="Terminal"
# 初始化一个新的 Languine 项目
languine init

# 这将：
# - 创建一个 languine.config.js 文件
# - 设置您的项目结构
# - 初始化必要的目录
```

### status

```bash title="Terminal"
# 检查您的翻译状态
languine status

# 显示：
# - 当前翻译进度
# - 缺失的翻译
# - 翻译统计信息
```

### validate

```bash title="Terminal"
# 验证您的翻译文件
languine validate

# 检查：
# - 文件结构
# - 缺失的键
# - 格式一致性
```

### export

```bash title="Terminal"
# 导出您的翻译
languine export

# 选项：
#   --format          导出格式（json, yaml 等）
#   --output         输出目录
```

### import

```bash title="Terminal"
# 从外部文件导入翻译
languine import

# 选项：
#   --source         源文件或目录
#   --format         导入格式（json, yaml 等）
```

## 全局选项

所有命令支持以下全局选项：

```bash title="Terminal"
# 所有命令可用的全局选项
languine [command] [options]

# 常见选项：
#   --help, -h       显示帮助信息
#   --version, -v    显示版本信息
#   --verbose        启用详细日志
#   --config         指定配置文件路径
```
