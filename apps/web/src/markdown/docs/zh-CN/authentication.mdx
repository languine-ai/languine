# 认证

要使用 Languine，您需要通过我们的引擎进行身份验证，开始只需几秒钟。

---

## 登录

您可以直接在 [Languine 控制面板](https://languine.ai/login) 上登录，或运行以下命令：

```bash title="终端"
npx languine@latest auth login
```


如果您已经有账户，系统将自动登录，并在您的机器上保存一个会话。

---

## 登出

您可以通过运行以下命令登出：

```bash title="终端"
npx languine@latest auth logout
```

---

## 用户会话

您可以通过运行以下命令检查您是否已登录：

```bash title="终端"
npx languine@latest auth whoami
```

---


# 环境变量

您可以设置以下环境变量来配置 Languine：

- `LANGUINE_API_KEY`：用于身份验证的 API 密钥。
- `LANGUINE_PROJECT_ID`：用于项目的项目 ID。

您可以在 [Languine 控制面板](https://languine.ai/login) 中找到它们。
