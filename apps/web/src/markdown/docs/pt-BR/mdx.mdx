# Markdown

Um arquivo markdown traduzido é tratado como um Documento e é traduzido como um todo para manter o contexto do documento.

## Ignorando conteúdo

Você pode ignorar o conteúdo entre as tags `<!-- languine-ignore-start -->` e `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Este conteúdo não será traduzido.
<!-- languine-ignore-end -->
```
