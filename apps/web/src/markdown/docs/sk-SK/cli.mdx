# CLI

Languine poskytuje CLI, ktoré vám pomôže preložiť vašu aplikáciu.

## Prehľad príkazov

### translate

Príkaz `translate` sa používa na preklad vašej aplikácie.

```bash title="Terminál"
# Začnite preklad vašej aplikácie
languine translate

# Možnosti:
#   --source-locale    Určte zdrojovú lokalitu (predvolené: en)
#   --target-locale    Určte cieľovú lokalitu(y), do ktorých sa má preložiť
#   --config          Cesta k vášmu súboru languine.config.js
```

### init

```bash title="Terminál"
# Inicializujte nový projekt Languine
languine init

# Toto vykoná:
# - Vytvorí súbor languine.config.js
# - Nastaví štruktúru vášho projektu
# - Inicializuje potrebné adresáre
```

### status

```bash title="Terminál"
# Skontrolujte stav vašich prekladov
languine status

# Zobrazuje:
# - Aktuálny pokrok v preklade
# - Chýbajúce preklady
# - Štatistiky prekladu
```

### validate

```bash title="Terminál"
# Overte svoje prekladové súbory
languine validate

# Kontroluje:
# - Štruktúru súboru
# - Chýbajúce kľúče
# - Konzistenciu formátu
```

### export

```bash title="Terminál"
# Exportujte svoje preklady
languine export

# Možnosti:
#   --format          Formát exportu (json, yaml, atď.)
#   --output         Výstupný adresár
```

### import

```bash title="Terminál"
# Importujte preklady z externých súborov
languine import

# Možnosti:
#   --source         Zdrojový súbor alebo adresár
#   --format         Formát importu (json, yaml, atď.)
```

## Globálne možnosti

Všetky príkazy podporujú nasledujúce globálne možnosti:

```bash title="Terminál"
# Globálne možnosti dostupné pre všetky príkazy
languine [príkaz] [možnosti]

# Spoločné možnosti:
#   --help, -h       Zobraziť informácie o pomoci
#   --version, -v    Zobraziť informácie o verzii
#   --verbose        Povoliť podrobné protokolovanie
#   --config         Určte cestu k súboru s konfiguráciou
```
