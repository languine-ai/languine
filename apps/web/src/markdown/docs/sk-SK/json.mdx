# JSON

Languine podporuje JSON súbory ako zdrojový formát s bežnými požiadavkami na pluralizáciu a interpoláciu. Tento formát sa používa v mnohých frameworkoch ako React, Vue, Ruby on Rails a ďalších.


---


## Nastavenie

Najprv sa uistite, že máte v koreňovom adresári projektu súbor s konfiguráciou languine.json alebo languine.ts. Tu je príklad:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Prekladanie

S nastavenou konfiguráciou spustite:

```bash
npx languine@latest translate
```

Keď spustíte tento príkaz, Languine:

- Načíta váš zdrojový JSON súbor (napr. locales/en.json)
- Zistí akékoľvek nové alebo upravené prekladové reťazce
- Vygeneruje preklady pre vaše cieľové jazyky
- Vytvorí alebo aktualizuje súbory JSON pre cieľový jazyk (napr. locales/sv.json, locales/de.json, locales/fr.json)
