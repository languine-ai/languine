# Autentizace

Pro použití Languine se musíte autentizovat s naším Enginem, začátek zabere jen pár sekund.

---

## Přihlášení

Můžete se přihlásit přímo na [Languine Dashboard](https://languine.ai/login) nebo spustit následující příkaz:

```bash title="Terminál"
npx languine@latest auth login
```


Pokud již máte účet, budete automaticky přihlášeni a relace bude uložena na vašem zařízení.

---

## Odhlášení

Můžete se odhlásit spuštěním následujícího příkazu:

```bash title="Terminál"
npx languine@latest auth logout
```

---

## Uživatelova relace

Můžete zkontrolovat, zda jste přihlášeni, spuštěním následujícího příkazu:

```bash title="Terminál"
npx languine@latest auth whoami
```

---


# Proměnné prostředí

Můžete nastavit následující proměnné prostředí pro konfiguraci Languine:

- `LANGUINE_API_KEY`: API klíč pro autentizaci.
- `LANGUINE_PROJECT_ID`: ID projektu pro použití v projektu.

Najdete je na [Languine Dashboard](https://languine.ai/login).
