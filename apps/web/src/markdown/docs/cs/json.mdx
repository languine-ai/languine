# JSON

Languine podporuje JSON soubory jako zdrojový formát s běžnými požadavky na pluralizaci a interpolaci. Tento formát se používá v mnoha rámcích, jako jsou React, Vue, Ruby on Rails a další.


---


## Nastavení

Nejprve se ujistěte, že máte v kořenovém adresáři projektu soubor s konfigurací languine.json nebo languine.ts. Zde je příklad:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## Překlad

S nastavenou konfigurací spusťte:

```bash
npx languine@latest translate
```

Když tuto příkaz spustíte, Languine:

- Načte váš zdrojový JSON soubor (např. locales/en.json)
- Detekuje jakékoli nové nebo upravené překladové řetězce
- Vygeneruje překlady pro vaše cílové jazyky
- Vytvoří nebo aktualizuje JSON soubory cílového jazyka (např. locales/sv.json, locales/de.json, locales/fr.json)
