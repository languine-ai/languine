# Rychlý start

Začněte instalací Languine.

Poté spusťte následující příkaz pro inicializaci Languine.

```bash title="Terminál"
npx languine@latest init
```

Postupujte podle kroků pro nastavení vašeho projektu.

## První překlad

Nyní jste připraveni přeložit svůj kód.

Začněte přidáním klíče do vašeho zdrojového souboru, například pokud se rozhodnete použít `en.json` jako váš zdrojový soubor, přidejte do něj klíč.

```json title="src/locales/en.json"
{
  "welcome": "Vítejte v naší aplikaci!",
  "user": {
    "greeting": "Ahoj, {{name}}!",
    "profile": {
      "title": "Váš profil",
      "edit": "Upravit profil"
    }
  },
  "notifications": {
    "messages": "Máte {{count}} novou zprávu(y).",
    "empty": "Žádné nové oznámení."
  }
}

```


## Překlad

Spusťte následující příkaz pro překlad vašeho obsahu:

```bash title="Terminál"
npx languine@latest translate
```

## To provede:
1. Vytvoří překladové soubory pro každý cílový jazyk
2. Vygeneruje vysoce kvalitní překlady pomocí Languine Engine
3. Vytvoří soubor `languine.lock` pro sledování stavů překladu

Soubor zámku pomáhá Languine identifikovat, které překlady potřebují aktualizaci, když se váš zdrojový obsah změní. Ujistěte se, že tento soubor zkomitujete do vašeho repozitáře.

## Další kroky

Nyní, když máte Languine nastavené, můžete:
- Přidat další jazyky do vašeho souboru `languine.json`
- Integrovat Languine do vašeho CI/CD pipeline
- Použít naše API pro programatické překlady

Podívejte se na naše další příručky, abyste se dozvěděli více o pokročilých funkcích Languine.
