# Markdown

Přeložený markdown soubor je považován za dokument a je přeložen jako celek, aby se zachoval kontext dokumentu.

## Ignorování obsahu

Můžete ignorovat obsah mezi značkami `<!-- languine-ignore-start -->` a `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Tento obsah nebude přeložen.
<!-- languine-ignore-end -->
```
