# அங்கீகாரம்

Languine ஐப் பயன்படுத்த, எங்கள் இயந்திரத்துடன் நீங்கள் அங்கீகரிக்க வேண்டும், தொடங்குவதற்கு சில விநாடிகள் மட்டுமே ஆகும்.

---

## உள்நுழைவு

நீங்கள் நேரடியாக [Languine Dashboard](https://languine.ai/login) இல் உள்நுழையலாம் அல்லது கீழ்காணும் கட்டளையை இயக்கலாம்:

```bash title="Terminal"
npx languine@latest auth login
```


நீங்கள் ஏற்கனவே ஒரு கணக்கைக் கொண்டிருந்தால், நீங்கள் தானாகவே உள்நுழைந்து, உங்கள் இயந்திரத்தில் ஒரு அமர்வு சேமிக்கப்படும்.

---

## வெளியேறு

நீங்கள் கீழ்காணும் கட்டளையை இயக்கி வெளியேறலாம்:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## பயனர் அமர்வு

நீங்கள் உள்நுழைந்துள்ளீர்களா என்பதை சரிபார்க்க கீழ்காணும் கட்டளையை இயக்கலாம்:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# சுற்றுச்சூழல் மாறிகள்

Languine ஐ கட்டமைக்க கீழ்காணும் சுற்றுச்சூழல் மாறிகளை நீங்கள் அமைக்கலாம்:

- `LANGUINE_API_KEY`: அங்கீகாரத்திற்காக பயன்படுத்த வேண்டிய API விசை.
- `LANGUINE_PROJECT_ID`: திட்டத்திற்காக பயன்படுத்த வேண்டிய திட்ட ID.

இவற்றை [Languine Dashboard](https://languine.ai/login) இல் காணலாம்.
