# JSON

Languine JSON கோப்புகளை பன்மை மற்றும் இடைமுகம் ஆகியவற்றிற்கான பொதுவான தேவைகளுடன் ஒரு மூல வடிவமாக ஆதரிக்கிறது. இந்த வடிவம் React, Vue, Ruby on Rails மற்றும் மேலும் பல போன்ற பல கட்டமைப்புகளில் பயன்படுத்தப்படுகிறது.


---


## அமைப்பு

முதலில், உங்கள் திட்டத்தின் அடிப்படையில் languine.json அல்லது languine.ts கட்டமைப்பு கோப்பு உள்ளதா என்பதை உறுதி செய்யவும். இதோ ஒரு எடுத்துக்காட்டு:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## மொழிபெயர்ப்பு

உங்கள் கட்டமைப்பு அமைக்கப்பட்ட பிறகு, இயக்கவும்:

```bash
npx languine@latest translate
```

இந்த கட்டளையை நீங்கள் இயக்கும் போது, Languine:

- உங்கள் மூல JSON கோப்பை ஏற்றும் (எடுத்துக்காட்டாக, locales/en.json)
- புதிய அல்லது மாற்றியமைக்கப்பட்ட மொழிபெயர்ப்பு சரங்களை கண்டறியும்
- உங்கள் இலக்கு மொழிகளுக்கான மொழிபெயர்ப்புகளை உருவாக்கும்
- இலக்கு மொழி JSON கோப்புகளை உருவாக்கும் அல்லது புதுப்பிக்கும் (எடுத்துக்காட்டாக, locales/sv.json, locales/de.json, locales/fr.json)
