# CI/CD

Languine உங்கள் பயன்பாட்டை மொழிபெயர்க்க CI/CD ஒருங்கிணைப்பை வழங்குகிறது.

---

## இது எப்படி வேலை செய்கிறது

Languine இல் CI/CD வேலைப்பாடு உங்கள் பயன்பாட்டில் மொழிபெயர்ப்பு புதுப்பிப்புகளை தானாகச் செய்ய ஒரு கட்டமைக்கப்பட்ட செயல்முறையை பின்பற்றுகிறது:

### 1. செயல்பாட்டின் ஆரம்பம்
- செயல்பாடு தொடங்கும் போது வேலைப்பாடு ஆரம்பிக்கிறது
- தேவையான Git கட்டமைப்பு மற்றும் அங்கீகாரங்களை அமைக்கிறது

### 2. Languine இயந்திரம் செயலாக்கம்
- Languine இயந்திரம் உங்கள் பயன்பாட்டின் உள்ளடக்கத்தை செயலாக்குகிறது
- கட்டமைக்கப்பட்ட அமைப்புகளைப் பயன்படுத்தி மொழிபெயர்ப்பு பணிகளை கையாள்கிறது

### 3. மாற்றங்கள் கண்டறிதல்
முறைமை எந்த மொழிபெயர்ப்பு மாற்றங்களையும் சரிபார்க்கிறது மற்றும் இரண்டு பாதைகளில் ஒன்றை பின்பற்றுகிறது:

**மாற்றங்கள் கண்டுபிடிக்கப்பட்டால்:**
- மாற்றங்கள் காப்பகத்திற்கு உறுதிப்படுத்தப்படுகின்றன
- மாற்றங்கள் குறிப்பிட்ட கிளைக்கு தள்ளப்படுகின்றன
- ஒரு Pull Request (PR) உருவாக்கப்படுகிறது

**மாற்றங்கள் கண்டுபிடிக்கப்படவில்லை என்றால்:**
- PR உருவாக்காமல் வேலைப்பாடு முடிகிறது

### 4. Pull Request உருவாக்கம்
மாற்றங்கள் கண்டறியப்பட்டால், Pull Request உருவாக்கப்படுகிறது:

**PR உள்ளடக்கம்:**
- மொழிபெயர்ப்பு புதுப்பிப்புகளை குறிக்கும் தெளிவான தலைப்பு
- மாற்றங்களின் விரிவான விளக்கம்
- தரத்திற்கான உறுதிப்படுத்தல்கள்
- மதிப்பீட்டிற்கான வழிமுறைகள்
- ஆவணங்கள் மற்றும் ஆதரவு இணைப்புகள்

**PR விளக்கம் உள்ளடக்கங்கள்:**
- மொழிபெயர்ப்பு புதுப்பிப்புகளின் சுருக்கம்
- என்ன மாற்றம் ஏற்பட்டது (புதுப்பிக்கப்பட்ட மொழிபெயர்ப்புகள், சொற்களின் ஒத்திசைவு)
- மதிப்பீட்டாளர்களுக்கான அடுத்த படிகள்
- ஆவணங்கள் மற்றும் ஆதரவு வளங்களுக்கு இணைப்புகள்
