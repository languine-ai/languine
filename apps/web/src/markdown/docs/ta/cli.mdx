# CLI

Languine உங்கள் பயன்பாட்டை மொழிபெயர்க்க உதவ ஒரு CLI-ஐ வழங்குகிறது.

## கட்டளைகள் மேலோட்டம்

### மொழிபெயர்ப்பு

`translate` கட்டளை உங்கள் பயன்பாட்டை மொழிபெயர்க்க பயன்படுத்தப்படுகிறது.

```bash title="Terminal"
# உங்கள் பயன்பாட்டின் மொழிபெயர்ப்பை தொடங்கவும்
languine translate

# விருப்பங்கள்:
#   --source-locale    மூல இடத்தை குறிப்பிடவும் (இயல்பாக: en)
#   --target-locale    மொழிபெயர்க்க வேண்டிய இலக்கு இடத்தை(களை) குறிப்பிடவும்
#   --config          உங்கள் languine.config.js கோப்பிற்கான பாதை
```

### ஆரம்பிக்கவும்

```bash title="Terminal"
# புதிய Languine திட்டத்தை ஆரம்பிக்கவும்
languine init

# இது:
# - ஒரு languine.config.js கோப்பை உருவாக்கும்
# - உங்கள் திட்ட அமைப்பை அமைக்கும்
# - தேவையான அடைவுகளை ஆரம்பிக்கும்
```

### நிலை

```bash title="Terminal"
# உங்கள் மொழிபெயர்ப்புகளின் நிலையை சரிபார்க்கவும்
languine status

# காட்டுகிறது:
# - தற்போதைய மொழிபெயர்ப்பு முன்னேற்றம்
# - காணாமல் போன மொழிபெயர்ப்புகள்
# - மொழிபெயர்ப்பு புள்ளிவிவரங்கள்
```

### சரிபார்க்கவும்

```bash title="Terminal"
# உங்கள் மொழிபெயர்ப்பு கோப்புகளை சரிபார்க்கவும்
languine validate

# சரிபார்க்கிறது:
# - கோப்பு அமைப்பு
# - காணாமல் போன விசைகள்
# - வடிவம் ஒத்திசைவு
```

### ஏற்றுமதி

```bash title="Terminal"
# உங்கள் மொழிபெயர்ப்புகளை ஏற்றுமதி செய்யவும்
languine export

# விருப்பங்கள்:
#   --format          ஏற்றுமதி வடிவம் (json, yaml, etc.)
#   --output         வெளியீட்டு அடைவு
```

### இறக்குமதி

```bash title="Terminal"
# வெளிப்புற கோப்புகளில் இருந்து மொழிபெயர்ப்புகளை இறக்குமதி செய்யவும்
languine import

# விருப்பங்கள்:
#   --source         மூல கோப்பு அல்லது அடைவு
#   --format         இறக்குமதி வடிவம் (json, yaml, etc.)
```

## உலகளாவிய விருப்பங்கள்

எல்லா கட்டளைகளும் கீழ்காணும் உலகளாவிய விருப்பங்களை ஆதரிக்கின்றன:

```bash title="Terminal"
# அனைத்து கட்டளைகளுக்கும் கிடைக்கும் உலகளாவிய விருப்பங்கள்
languine [command] [options]

# பொதுவான விருப்பங்கள்:
#   --help, -h       உதவி தகவலைக் காண்பிக்கவும்
#   --version, -v    பதிப்பு தகவலைக் காண்பிக்கவும்
#   --verbose        விரிவான பதிவு செயல்படுத்தவும்
#   --config         கட்டமைப்பு கோப்பின் பாதையை குறிப்பிடவும்
```
