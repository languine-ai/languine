# மார்க்டவுன்

ஒரு மொழிபெயர்க்கப்பட்ட மார்க்டவுன் கோப்பு ஒரு ஆவணமாகக் கருதப்படுகிறது, மேலும் ஆவணத்தின் சூழலைக் காக்க முழுமையாக மொழிபெயர்க்கப்படுகிறது.
