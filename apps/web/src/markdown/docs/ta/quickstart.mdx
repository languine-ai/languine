# விரைவான தொடக்கம்

Languine ஐ நிறுவுவதன் மூலம் தொடங்குங்கள்.

பின்னர், Languine ஐ ஆரம்பிக்க கீழ்காணும் கட்டளையை இயக்குங்கள்.

```bash title="Terminal"
npx languine@latest init
```

உங்கள் திட்டத்தை அமைக்க படிகளை பின்பற்றவும்.

## முதல் மொழிபெயர்ப்பு

இப்போது நீங்கள் உங்கள் குறியீட்டை மொழிபெயர்க்க தயாராக இருக்கிறீர்கள்.

உங்கள் மூல கோப்பிற்கு ஒரு விசையை சேர்க்க ஆரம்பிக்கவும், எடுத்துக்காட்டாக நீங்கள் `en.json` ஐ உங்கள் மூல கோப்பாக பயன்படுத்த விரும்பினால், அதில் ஒரு விசையை சேர்க்கவும்.

```json title="src/locales/en.json"
{
  "welcome": "எங்கள் பயன்பாட்டிற்கு வரவேற்கிறோம்!",
  "user": {
    "greeting": "வணக்கம், {{name}}!",
    "profile": {
      "title": "உங்கள் சுயவிவரம்",
      "edit": "சுயவிவரத்தை திருத்தவும்"
    }
  },
  "notifications": {
    "messages": "உங்களுக்கு {{count}} புதிய செய்தி(கள்) உள்ளன.",
    "empty": "புதிய அறிவிப்புகள் இல்லை."
  }
}

```


## மொழிபெயர்க்கவும்

உங்கள் உள்ளடக்கத்தை மொழிபெயர்க்க கீழ்காணும் கட்டளையை இயக்குங்கள்:

```bash title="Terminal"
npx languine@latest translate
```

## இது செய்யும்:
1. ஒவ்வொரு இலக்கு மொழிக்கான மொழிபெயர்ப்பு கோப்புகளை உருவாக்கவும்
2. Languine Engine மூலம் உயர்தர மொழிபெயர்ப்புகளை உருவாக்கவும்
3. மொழிபெயர்ப்பு நிலைகளை கண்காணிக்க `languine.lock` கோப்பை உருவாக்கவும்

இந்த லாக் கோப்பு, உங்கள் மூல உள்ளடக்கம் மாறும் போது, எந்த மொழிபெயர்ப்புகள் புதுப்பிக்க வேண்டும் என்பதை Languine ஐ அடையாளம் காண உதவுகிறது. இந்த கோப்பை உங்கள் களஞ்சியத்தில் சேர்க்க உறுதியாகவும்.

## அடுத்த படிகள்

இப்போது நீங்கள் Languine ஐ அமைத்துள்ளதால், நீங்கள்:
- உங்கள் `languine.json` கோப்பில் மேலும் மொழிகளைச் சேர்க்கலாம்
- உங்கள் CI/CD குழாயில் Languine ஐ ஒருங்கிணைக்கலாம்
- நிரலாக்க மொழிபெயர்ப்புகளுக்கு எங்கள் API ஐப் பயன்படுத்தலாம்

Languine இன் மேம்பட்ட அம்சங்களைப் பற்றி மேலும் அறிய எங்கள் பிற வழிகாட்டிகளைப் பார்க்கவும்.
