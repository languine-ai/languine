# Rýchly štart

Začnite inštaláciou Languine.

Potom spustite nasledujúci príkaz na inicializáciu Languine.

```bash title="Terminál"
npx languine@latest init
```

Postupujte podľa krokov na nastavenie vášho projektu.

## Prvý preklad

Teraz ste pripravení preložiť svoj kód.

Začnite pridaním kľúča do vášho zdrojového súboru, napríklad ak sa rozhodnete použiť `en.json` ako váš zdrojový súbor, pridajte do neho kľúč.

```json title="src/locales/en.json"
{
  "welcome": "Vitajte v našej aplikácii!",
  "user": {
    "greeting": "Ahoj, {{name}}!",
    "profile": {
      "title": "Váš profil",
      "edit": "Upraviť profil"
    }
  },
  "notifications": {
    "messages": "Máte {{count}} novú správu/správy.",
    "empty": "Žiadne nové oznámenia."
  }
}

```


## Preložiť

Spustite nasledujúci príkaz na preloženie vášho obsahu:

```bash title="Terminál"
npx languine@latest translate
```

## Toto urobí:
1. Vytvorí prekladové súbory pre každý cieľový jazyk
2. Vygeneruje kvalitné preklady pomocou Languine Engine
3. Vytvorí súbor `languine.lock` na sledovanie stavov prekladu

Súbor s uzamknutím pomáha Languine identifikovať, ktoré preklady potrebujú aktualizácie, keď sa váš zdrojový obsah zmení. Uistite sa, že tento súbor uložíte do svojho repozitára.

## Ďalšie kroky

Teraz, keď máte Languine nastavené, môžete:
- Pridať ďalšie jazyky do vášho súboru `languine.json`
- Integrovať Languine do vášho CI/CD pipeline
- Použiť naše API na programatické preklady

Pozrite si naše ďalšie príručky, aby ste sa dozvedeli viac o pokročilých funkciách Languine.
