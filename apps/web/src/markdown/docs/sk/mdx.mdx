# Markdown

Preložený markdown súbor sa považuje za dokument a prekladá sa ako celok, aby sa zachoval kontext dokumentu.

## Ignorovanie obsahu

Môžete ignorovať obsah medzi značkami `<!-- languine-ignore-start -->` a `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Tento obsah nebude preložený.
<!-- languine-ignore-end -->
```
