# Autentifikácia

Na používanie Languine sa musíte autentifikovať s naším Engine, začatie trvá len niekoľko sekúnd.

---

## Prihlásenie

Môžete sa prihlásiť priamo na [Languine Dashboard](https://languine.ai/login) alebo spustiť nasledujúci príkaz:

```bash title="Terminál"
npx languine@latest auth login
```


Ak už máte účet, automaticky sa prihlásite a relácia sa uloží vo vašom zariadení.

---

## Odhlásenie

Môžete sa odhlásiť spustením nasledujúceho príkazu:

```bash title="Terminál"
npx languine@latest auth logout
```

---

## Užívateľská relácia

Môžete skontrolovať, či ste prihlásení, spustením nasledujúceho príkazu:

```bash title="Terminál"
npx languine@latest auth whoami
```

---


# Premenné prostredia

Môžete nastaviť nasledujúce premenné prostredia na konfiguráciu Languine:

- `LANGUINE_API_KEY`: API kľúč na autentifikáciu.
- `LANGUINE_PROJECT_ID`: ID projektu na použitie pre projekt.

Nájdete ich na [Languine Dashboard](https://languine.ai/login).
