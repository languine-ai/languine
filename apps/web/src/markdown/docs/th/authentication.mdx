# การตรวจสอบสิทธิ์

ในการใช้ Languine คุณต้องตรวจสอบสิทธิ์กับ Engine ของเรา ใช้เวลาเพียงไม่กี่วินาทีในการเริ่มต้น

---

## ลงชื่อเข้าใช้

คุณสามารถลงชื่อเข้าใช้ได้โดยตรงที่ [Languine Dashboard](https://languine.ai/login) หรือรันคำสั่งต่อไปนี้:

```bash title="Terminal"
npx languine@latest auth login
```


หากคุณมีบัญชีอยู่แล้ว คุณจะถูกลงชื่อเข้าใช้โดยอัตโนมัติและเซสชันจะถูกบันทึกไว้ในเครื่องของคุณ

---

## ลงชื่อออก

คุณสามารถลงชื่อออกโดยรันคำสั่งต่อไปนี้:

```bash title="Terminal"
npx languine@latest auth logout
```

---

## เซสชันผู้ใช้

คุณสามารถตรวจสอบว่าคุณได้ลงชื่อเข้าใช้หรือไม่โดยรันคำสั่งต่อไปนี้:

```bash title="Terminal"
npx languine@latest auth whoami
```

---


# ตัวแปรสภาพแวดล้อม

คุณสามารถตั้งค่าตัวแปรสภาพแวดล้อมต่อไปนี้เพื่อกำหนดค่า Languine:

- `LANGUINE_API_KEY`: คีย์ API ที่ใช้สำหรับการตรวจสอบสิทธิ์
- `LANGUINE_PROJECT_ID`: รหัสโครงการที่ใช้สำหรับโครงการ

คุณสามารถหาข้อมูลเหล่านี้ได้ใน [Languine Dashboard](https://languine.ai/login).
