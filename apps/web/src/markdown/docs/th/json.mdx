# JSON

Languine รองรับไฟล์ JSON เป็นรูปแบบแหล่งข้อมูลที่มีความต้องการทั่วไปสำหรับการทำให้เป็นพหูพจน์และการแทรกค่า รูปแบบนี้ถูกใช้ในหลายเฟรมเวิร์ก เช่น React, Vue, Ruby on Rails และอื่นๆ


---


## การตั้งค่า

ก่อนอื่นให้แน่ใจว่าคุณมีไฟล์กำหนดค่า languine.json หรือ languine.ts ในรากโปรเจกต์ของคุณ นี่คือตัวอย่าง:

```json
{
  "locale": {
    "source": "en",
    "targets": ["sv", "de", "fr"]
  },
  "files": {
    "json": {
      "include": ["locales/[locale].json"]
    }
  }
}

```

## การแปล

เมื่อคุณตั้งค่ากำหนดค่าเสร็จแล้ว ให้รัน:

```bash
npx languine@latest translate
```

เมื่อคุณรันคำสั่งนี้ Languine จะ:

- โหลดไฟล์ JSON แหล่งข้อมูลของคุณ (เช่น locales/en.json)
- ตรวจจับสตริงการแปลใหม่หรือที่ถูกแก้ไข
- สร้างการแปลสำหรับภาษาที่คุณต้องการ
- สร้างหรืออัปเดตไฟล์ JSON ของภาษาที่คุณต้องการ (เช่น locales/sv.json, locales/de.json, locales/fr.json)
