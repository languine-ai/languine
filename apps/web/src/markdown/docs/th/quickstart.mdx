# เริ่มต้นอย่างรวดเร็ว

เริ่มต้นด้วยการติดตั้ง Languine.

จากนั้น รันคำสั่งต่อไปนี้เพื่อเริ่มต้น Languine.

```bash title="Terminal"
npx languine@latest init
```

ทำตามขั้นตอนเพื่อจัดตั้งโปรเจกต์ของคุณ.

## การแปลครั้งแรก

ตอนนี้คุณพร้อมที่จะแปลโค้ดของคุณแล้ว.

เริ่มต้นด้วยการเพิ่มคีย์ไปยังไฟล์ต้นฉบับของคุณ เช่น หากคุณเลือกใช้ `en.json` เป็นไฟล์ต้นฉบับ ให้เพิ่มคีย์ไปยังไฟล์นั้น.

```json title="src/locales/en.json"
{
  "welcome": "ยินดีต้อนรับสู่แอปพลิเคชันของเรา!",
  "user": {
    "greeting": "สวัสดี, {{name}}!",
    "profile": {
      "title": "โปรไฟล์ของคุณ",
      "edit": "แก้ไขโปรไฟล์"
    }
  },
  "notifications": {
    "messages": "คุณมีข้อความใหม่ {{count}} ข้อความ.",
    "empty": "ไม่มีการแจ้งเตือนใหม่."
  }
}

```


## แปล

รันคำสั่งต่อไปนี้เพื่อแปลเนื้อหาของคุณ:

```bash title="Terminal"
npx languine@latest translate
```

## สิ่งนี้จะ:
1. สร้างไฟล์การแปลสำหรับแต่ละภาษาที่ต้องการ
2. สร้างการแปลคุณภาพสูงด้วย Languine Engine
3. สร้างไฟล์ `languine.lock` เพื่อติดตามสถานะแปล

ไฟล์ล็อคช่วยให้ Languine ระบุการแปลที่ต้องการการอัปเดตเมื่อเนื้อหาต้นฉบับของคุณมีการเปลี่ยนแปลง อย่าลืมบันทึกไฟล์นี้ในที่เก็บข้อมูลของคุณ.

## ขั้นตอนถัดไป

ตอนนี้ที่คุณตั้งค่า Languine เสร็จแล้ว คุณสามารถ:
- เพิ่มภาษามากขึ้นในไฟล์ `languine.json`
- รวม Languine เข้ากับ CI/CD pipeline ของคุณ
- ใช้ API ของเราเพื่อการแปลแบบโปรแกรม

ตรวจสอบคู่มืออื่น ๆ ของเราเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับฟีเจอร์ขั้นสูงของ Languine.
