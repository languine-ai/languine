# Markdown

ไฟล์ markdown ที่แปลแล้วจะถูกจัดการเป็นเอกสาร และจะถูกแปลทั้งหมดเพื่อรักษาบริบทของเอกสาร

## การละเว้นเนื้อหา

คุณสามารถละเว้นเนื้อหาระหว่างแท็ก `<!-- languine-ignore-start -->` และ `<!-- languine-ignore-end -->` ได้

```mdx
<!-- languine-ignore-start -->
เนื้อหานี้จะไม่ถูกแปล
<!-- languine-ignore-end -->
```
