# Markdown

## การละเว้นเนื้อหา

คุณสามารถละเว้นเนื้อหาระหว่างแท็ก `<!-- languine-ignore-start -->` และ `<!-- languine-ignore-end -->`

```mdx
<!-- languine-ignore-start -->
เนื้อหานี้จะไม่ถูกแปล
<!-- languine-ignore-end -->
```
