# Nadpisy

Languine pozwala na nadpisywanie tłumaczeń dla konkretnych kluczy. Jest to przydatne, gdy chcesz zapewnić, że konkretne tłumaczenie jest używane dla danego klucza, niezależnie od domyślnego tłumaczenia.

## Jak to działa

W panelu sterowania możesz tworzyć nadpisania dla konkretnych kluczy. Możesz również tworzyć nadpisania dla konkretnych języków.
