## Kluczowe funkcje

- **Tłumaczenia wspomagane przez AI**: Wykorzystaj zaawansowane modele uczenia maszynowego do generowania wysokiej jakości tłumaczeń w ponad 100 językach
- **Integracja z Gitem**: Synchronizuj swoje tłumaczenia bezpośrednio z repozytoriami Git i automatyzuj aktualizacje za pomocą potoków CI/CD
- **Skupienie na programistach**: Stworzony z myślą o programistach, oferujący potężne CLI, API i integrację z GitHub Action
- **Niestandardowe parsery**: Wsparcie dla różnych formatów plików oraz możliwość tworzenia niestandardowych parserów dostosowanych do Twoich potrzeb
- **Kontrola wersji**: Śledź zmiany, zarządzaj gałęziami i utrzymuj historię tłumaczeń
- **Open Source**: W pełni otwarte źródło dostępne na GitHubie, co zapewnia przejrzystość i wkład społeczności

## Jak to działa

1. **Konfiguracja**: Zainstaluj Languine CLI i skonfiguruj swój projekt za pomocą prostego procesu inicjalizacji
2. **Połączenie**: Połącz swoje repozytorium i ustaw preferowany proces pracy (CLI, API lub GitHub Action)
3. **Tłumaczenie**: Prześlij swoje źródłowe treści i pozwól Languine zająć się tłumaczeniami na docelowe języki
4. **Synchronizacja**: Automatycznie synchronizuj przetłumaczone treści z powrotem do swojego repozytorium i wdrażaj z pewnością

## Jak zacząć

Aby rozpocząć korzystanie z Languine, postępuj zgodnie z naszym przewodnikiem [Instalacja](/docs/installation), aby skonfigurować CLI i stworzyć swój pierwszy projekt. Następnie zapoznaj się z przewodnikiem [Szybki start](/docs/quickstart), aby poznać podstawy zarządzania tłumaczeniami za pomocą Languine.

Aby uzyskać szczegółowe informacje na temat konkretnych funkcji i zaawansowanego użytkowania, zapoznaj się z naszymi obszernymi sekcjami dokumentacji:

- [Konfiguracja](/docs/configuration) - Dowiedz się, jak skonfigurować Languine dla swojego projektu
- [CLI](/docs/cli) - Odkryj pełny zakres poleceń CLI
- [API](/docs/api) - Zintegruj Languine programowo
- [Przepływy pracy](/docs/workflows) - Ustaw automatyczne przepływy pracy tłumaczeń
