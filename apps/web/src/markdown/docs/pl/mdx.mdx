# Markdown

Przetłumaczony plik markdown jest traktowany jako dokument i jest tłumaczony w całości, aby zachować kontekst dokumentu.

## Ignorowanie treści

Możesz zignorować treść pomiędzy znacznikami `<!-- languine-ignore-start -->` i `<!-- languine-ignore-end -->`.

```mdx
<!-- languine-ignore-start -->
Ta treść nie będzie tłumaczona.
<!-- languine-ignore-end -->
```
